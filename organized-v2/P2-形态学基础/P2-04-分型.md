# P2-04 分型（顶/底分型的定义与心理分析）

> **文件用途说明**：本文件是面向 AI 系统的缠论知识点文件，内容须自洽、无歧义、可机器解析。
> 所有定义优先使用充要条件格式，经验性判断须单独标注并与精确定义区分。

## 概述

分型是缠论形态学的最小构件，也是笔、线段、中枢等所有上层结构递归定义的启始程序。
本章定义顶分型与底分型（均以经包含关系处理后的相邻三根 K 线为操作对象），并阐明包含关系处理规则、分型的客观唯一性及其心理机制。
原文在课62（初版）至课65（再说明）建立基准定义，课77/81/82 为补充说明，以课62+课65 综合为最终有效基准。

## 📖 定义

### 定义一：包含关系

定义：当相邻两根 K 线（第 i 根、第 i+1 根）满足以下任一条件时，构成包含关系，需进行合并处理；不满足则保持独立：
- 条件 A（正向包含）：`g_i >= g_{i+1}` AND `d_i <= d_{i+1}`（第 i 根完全覆盖第 i+1 根）
- 条件 B（反向包含）：`g_i <= g_{i+1}` AND `d_i >= d_{i+1}`（第 i+1 根完全覆盖第 i 根）

**合并规则（方向相关）**：
- 向上时：合并为 `[max(d_i, d_{i+1}), max(g_i, g_{i+1})]`
- 向下时：合并为 `[min(d_i, d_{i+1}), min(g_i, g_{i+1})]`

其中 `g_i` 为第 i 根 K 线最高点，`d_i` 为最低点。

> 原文引用："在向上时，把两K线的最高点当高点，而两K线低点中的较高者当成低点，这样就把两K线合并成一新的K线；反之，当向下时，把两K线的最低点当低点，而两K线高点中的较低者当成高点。"
>
> ——课62（0567）

**方向的几何判断**（用于确定向上或向下）：

设第 n 根 K 线与第 n+1 根构成包含关系，而第 n 根与第 n-1 根不是包含关系：
- 若 `g_n >= g_{n-1}`，则称第 n-1、n、n+1 根 K 线**向上**
- 若 `d_n <= d_{n-1}`，则称第 n-1、n、n+1 根 K 线**向下**

> 原文引用："假设，第n根K线满足第n根与第n+1根的包含关系，而第n根与第n-1根不是包含关系，那么如果gn>=gn-1，那么称第n-1、n、n+1根K线是向上的；如果dn<=dn-1，那么称第n-1、n、n+1根K线是向下的。"
>
> ——课65（0593）

**顺序原则（包含关系不满足传递律）**：先用第 1、2 根合并为新 K 线，再用新 K 线与第 3 根比较；第 1、3 根不可直接判断包含关系。

### 定义二：顶分型

定义：当且仅当经包含关系处理后的相邻三根 K 线（左、中、右），满足以下全部条件时，构成**顶分型**；不满足任一条件则不构成顶分型：
- 条件 A：中间 K 线最高点 `g_mid` > 左 K 线最高点 `g_left`
- 条件 B：中间 K 线最高点 `g_mid` > 右 K 线最高点 `g_right`

顶分型的**顶**（简称"顶"）= 中间 K 线的最高点 `g_mid`。

> 原文引用："顶分型：相邻三根K线，中间那根K线的最高点比左右两根K线的最高点都高。"
>
> ——课62（0567）

---

### 定义三：底分型

定义：当且仅当经包含关系处理后的相邻三根 K 线（左、中、右），满足以下全部条件时，构成**底分型**；不满足任一条件则不构成底分型：
- 条件 A：中间 K 线最低点 `d_mid` < 左 K 线最低点 `d_left`
- 条件 B：中间 K 线最低点 `d_mid` < 右 K 线最低点 `d_right`

底分型的**底**（简称"底"）= 中间 K 线的最低点 `d_mid`。

> 原文引用："图2这种叫底分型，第二K线低点是相邻三K线低点中最低的，而高点也是相邻三K线高点中最低的。"
>
> ——课62（0567）

---

### 定义四：分型的客观唯一性

定义：当且仅当被分析的 K 线已经走出来（历史已定），对给定品种、给定周期的 K 线图，按上述定义处理所得到的全部分型，是**唯一**且**客观**的——与分析时间无关，与分析人无关，不同时间分析同一已完成 K 线得到相同结果。

未走出的 K 线（当下未完成的最后一根 K 线）不参与分型判断。

> 原文引用："这种定义是唯一的，有统一答案的，就算是本ID，如果弄错了，也就是错，没有任何含糊的地方，是可以在当下或任何时候明确无误地给出唯一答案的，这答案与时间无关，与人无关，是客观的，不可更改的。"
>
> ——课65（0593）

---

### 定义五：分型的当下可修改性

定义：当某级别当下的 K 线尚未走完时，当前末尾分型的判断可能随新 K 线的走出而发生改变（前一根 K 线可能被新 K 线通过包含处理合并，导致分型消失或改变）。此特性称为**分型的当下可修改性**。

一旦分型的三根组成 K 线全部完成，该分型不可修改。

> 原文引用："分型可修改，证明图形没完成。……在理论的框架下，只有极少的可能，而这些可能，就成为综合判断的关键条件。"
>
> ——课69（0627）

## 📐 定理

### 定理一：顶必为顶分型，底必为底分型

前提：[在任何级别的任何走势中，某时间点被确认为该级别的顶（局部最高点）]
→ 结论：[该顶必然对应一个顶分型；反之，无顶分型则无顶]

前提：[在任何级别的任何走势中，某时间点被确认为该级别的底（局部最低点）]
→ 结论：[该底必然对应一个底分型；反之，无底分型则无底]

> 原文引用："所有的顶必须是顶分型的，反之，所以的底都是底分型的。……没有顶分型，没有顶；反之，没有底分型，没有底。"
>
> ——课81（0695）

**操作推论**：在操作级别 K 线图上，若尚无顶分型，可持有不卖，等顶分型出现再操作。

---

### 定理二：分型是多空三次较量的结构结果

前提：[出现顶分型（经包含处理的标准三 K 线模型）]
→ 结论：[该顶分型的形成经历了买方三次攻击均被卖方击败的过程：第一根 K 线高点被卖方阻击产生回落；第二根 K 线出现更高高点但在小级别出现力度背驰（形成上影）；第三根 K 线买方再次攻击被完全击败，不能创新高，在小级别呈现第二类卖点走势]

> 原文引用："一个分型结构的出现，如同中枢，都是经过一个三次的反复心理较量过程，只是中枢用的是三个次级别。所谓一而再、再而三，三而竭，所以一个顶分型就这样出现了。"
>
> ——课82（0708）

---

### 定理三：分型的自相似性

前提：[分型结构在任意级别（1分钟图、日线图、月线图）均可出现]
→ 结论：[分型在不同级别上具有结构同构性（自相似性），级别仅影响分型所对应的市场含义的大小和持续时间，不影响定义本身]

> 原文引用："分型、笔、线段，在1分钟图上可以分辨，在月线图上的道理是一样的。"
>
> ——课69（0627）

---

### 定理四：分型是中枢递归定义的启始程序

前提：[需要对某级别走势建立中枢、走势类型等上层结构]
→ 结论：[分型是该递归链条的起始环节：分型 → 笔 → 线段 → 最小级别中枢 → 各级别中枢/走势类型；分型本身并非不可替代，其作用是保证上层定义的唯一性和可操作性]

> 原文引用："分型只是中枢与走势级别递归定义的一个启始程序，甚至可以说，并不是本ID理论中必然需要的东西，其目的，不过是为了中枢等的递归性定义中给出其最开始的部分。"
>
> ——课84（0718）

### 包含关系处理的判断树

遇到相邻两 K 线需要判断是否存在包含关系 →

```
步骤1：比较两 K 线的高低点范围
  ├─ g_i >= g_{i+1} AND d_i <= d_{i+1}？
  │   是 → 正向包含，需合并
  │   否 →
  ├─ g_i <= g_{i+1} AND d_i >= d_{i+1}？
  │   是 → 反向包含，需合并
  │   否 → 无包含关系，独立处理，进入步骤2

步骤2（需合并时）：判断当前方向
  ├─ 向上（g_n >= g_{n-1}）？
  │   是 → 合并为 [max(d_i,d_{i+1}), max(g_i,g_{i+1})]
  │   否 → 向下，合并为 [min(d_i,d_{i+1}), min(g_i,g_{i+1})]

步骤3：合并后继续与下一根 K 线比较（顺序原则，逐根处理）
```

**注意**：包含关系不满足传递律。第1、2根有包含关系，第2、3根有包含关系，不意味着第1、3根有包含关系——必须先处理第1、2，再用合并后的结果处理第3。

### 中继分型 vs 转折分型的判断树

遇到一个顶分型（或底分型），需要判断是中继还是转折 →

```
判断分型出现后是否延伸为笔：

条件A（强杀伤力信号）：
  第三根 K 线跌破第一根 K 线的底
  AND 第三根 K 线不能高收到第一根 K 线区间的一半之上
  → 判断倾向：转折（杀伤力强，延伸为笔的概率极大）

条件B（弱杀伤力信号）：
  第一根 K 线是长阳线（或长阴线），第二、三根均为小阴小阳
  → 判断倾向：中继（成为真正顶的可能性很小）

条件C（辅助验证——小级别走势）：
  分型成立后对应的小级别中枢是否出现第三类卖点（且不形成中枢扩张）？
  是 → 几乎100%将延伸为笔（转折）
  否 →
    小级别是否出现背驰或盘整背驰？
    是 → 判断倾向：中继
    否 → 继续观察

条件D（连续中继分型的衰减规律）：
  第一个中继分型后小级别已出现背驰或盘整背驰？
  是 → 下一个分型中继可能性大幅降低，接近转折
```

> 原文引用："要不出现笔并最终有效破坏该顶分型，那一定要出现某级别的第三类买点，否则就算有短时间的新高，也一定是假突破。"
>
> ——课82（0708）/ 课79（0682）

### 分型 vs 分形（数学分形理论）的区分

遇到需要区分"分型"与"分形（fractal theory）"→

| 维度 | 缠论分型 | 数学分形理论 |
|------|---------|------------|
| 来源 | 直接来源于市场走势本身的逻辑 | 先验数学理论硬套到市场 |
| 唯一性 | 充要条件定义，结果唯一 | 归纳性，划分不唯一 |
| 结论性质 | 在完全分类基础上的绝对结论 | 有缺陷的归纳性结论 |

> 原文引用："分型与分形有着本质的不同，本ID所说的分型，是建立在一个K线组合的纯粹分类的基础上，任何与这个纯粹分类不同的，都必然是错误的。"
>
> ——课84（0718）

### 三相邻 K 线的完全分类

经包含关系处理后，三相邻无包含关系的 K 线只有以下四种组合（来自课62图7）：

| 类型 | 条件（经包含处理后） | 分型结论 |
|------|------|---------|
| 上升 K 线 | 左→中：升；中→右：升（`g_left < g_mid < g_right` AND `d_left < d_mid < d_right`） | 不构成分型 |
| **顶分型** | 左→中：升；中→右：降（`g_mid > g_left` AND `g_mid > g_right`） | ✅ 构成顶分型 |
| 下降 K 线 | 左→中：降；中→右：降（`g_left > g_mid > g_right` AND `d_left > d_mid > d_right`） | 不构成分型 |
| **底分型** | 左→中：降；中→右：升（`d_mid < d_left` AND `d_mid < d_right`） | ✅ 构成底分型 |

注："升"指相邻 K 线的高点和低点均递增；"降"指均递减。四类互斥穷举，构成完全分类。

## 💡 操作规则

### 操作规则一：分型辅助止盈/止损操作（日线级别以上）

```
前提：[已在某级别持有头寸] AND [操作级别 K 线图上出现顶分型（多头持仓）或底分型（空头持仓）]
触发：[分型的三根组成 K 线全部完成走出]
操作：
  1. 记录分型位置，进入小级别（5分钟或1分钟）观察
  2. 若小级别对应中枢出现第三类卖点且不形成中枢扩张 → 100%确认为转折，执行减仓/清仓
  3. 若小级别出现背驰或盘整背驰 → 判断倾向为中继，可回补或继续持有
  4. 若两者均无信号 → 继续持有，等待更多 K 线确认
冲突优先级：当分型信号与更高级别走势判断冲突时，以更高级别结构判断优先；
  分型操作须配合小级别验证，不可单独依赖操作级别 K 线图上的分型形态。
```

---

### 操作规则二：分型"当下性"处置规则

```
前提：[当前操作级别最后一根 K 线尚未走完（正在运行中）]
触发：[基于当前末尾 K 线判断出现分型]
操作：
  1. 标注为"待确认分型"，不以此为操作依据
  2. 等待该 K 线完成（时间周期结束）
  3. 该 K 线完成后 → 重新按定义判断，若分型成立 → 转为"已确认分型"，可进行操作判断
冲突优先级：未完成 K 线形成的临时分型不可触发任何操作规则；
  所有分型判断必须基于已完成的 K 线。
```

---

### 操作规则三：无顶分型不卖出原则

```
前提：[持有多头仓位] AND [操作级别 K 线图上未出现顶分型]
触发：[价格连续上涨，无任何顶分型信号]
操作：持有，不卖出 → 等待操作级别出现顶分型 → 出现后进入操作规则一判断
冲突优先级：若更高级别出现明确背驰信号，则该原则让位于更高级别的背驰操作判断。
```

## 📜 经验判断

> ⚠️ **原文未给出充要条件**，以下为原文经验性指引，AI 使用时须视为参考倾向而非确定规则。

| 场景 | 判断倾向（原文指引） | 边界（不适用情形） |
|------|-------------------|--------------------|
| 顶分型中第一根为长阳线，第二、三根均为小阴小阳 | 中继分型可能性大，成为真正顶概率很小 | 若小级别已出现第三类卖点则不适用此倾向 |
| 顶分型第三根 K 线跌破第一根底且不能收回一半以上 | 转折分型，杀伤力强，延伸为笔概率极大 | 若市场出现某级别强势第三类买点则可能反转 |
| 中继顶分型后小级别已出现背驰或盘整背驰 | 下一顶分型转折可能性大幅增加 | 首次湿吻后的第一个分型不适用此规律 |
| 包含关系中第一根阳线被第二根长阴完全吃掉 | 最坏一种包含关系，转折力度强 | 需结合更高级别走势结构综合判断 |
| 顶分型后是否有效跌破5日均线 | 未跌破5日线 → 调整有限（中继可能）；有效跌破 → 较大调整 | 均线辅助工具，不能替代分型定义的充要条件判断 |
| 日线图上与5分钟图结合操作 | 日线中枢震荡无顶分型 → 不操作/少做短差；顶分型出现 → 对应5分钟第一、二类卖点 | 仅适用于日线与5分钟的时间尺度组合，其他尺度需相应调整 |

![顶底分型示意图](../../108/pic/0567.jpg)

> 原文对此图说明："先看图中的第1、2，图中的小线段代表的是K线，这里不分阳线阴线，只看K线高低点。像图1这种，第二K线高点是相邻三K线高点中最高的，而低点也是相邻三K线低点中最高的，本ID给一个定义叫顶分型；图2这种叫底分型。"
>
> ——课62（0567）

### 图示二：包含关系处理与三 K 线完全分类（课62原图）

![三K线完全分类与包含关系处理](../../108/pic/0567.jpg)

> 原文对此图说明（图7）："经过以上处理，没有包含关系的图形中，三相邻K线之间可能组合的一个完全分类，其中的二、四，就是分别是顶分型和底分型，一可以叫上升K线，三可以叫下降K线。"
>
> ——课62（0567）
