# P1-02-当下性哲学（核心操作思维）

> **文件用途说明**：本文件是面向 AI 系统的缠论知识点文件，内容须自洽、无歧义、可机器解析。
> 所有定义优先使用充要条件格式，经验性判断须单独标注并与精确定义区分。

## 概述

当下性哲学是缠论 P1-认知基础与交易哲学中的核心操作思维，贯穿整个交易体系的认识论基础。
本章定义了价格唯一性公设、完全分类操作原则和分段函数操作模型，为所有技术分析操作提供认识论前提与执行框架。
原文从课04至课098持续深化；课084以"破当下执"对当下性本身作出修正，是本章最终版本的补充说明。

## 📖 定义

### 定义一：价格唯一性公设

定义：当 [分析依据为市场实际成交价格直接形成的走势] 且 [不引入价格之外的信息（基本面、消息面、政策面预期等）作为操作触发条件] 时，称为符合价格唯一性的缠论分析；不满足任一条件（特别是以价格之外的信息直接触发操作）则不构成缠论体系内的有效操作依据。

> 原文引用："站在纯交易的角度，价格只有当下，当下只有价格，除了价格与依据时间延伸出来的走势，市场的任何其他东西都是可以忽略不计的。"
>
> ——课030（0459）

---

### 定义二：完全分类操作原则

定义：当 [对当前走势的所有后续可能状态进行穷举性分类（不排除任何可能性）] 且 [为每种分类状态预设对应的操作指令（买入/卖出/持有）] 且 [操作指令的触发完全依赖走势实际到达某一边界条件] 时，称为完全分类操作；不满足任一条件（特别是存在"假设某特定情况先成立，再推断操作"的预测成分）则不构成完全分类操作。

> 原文引用："对于一个真正的操作者，没有任何情况是意外的，因为，所有的情况都被完全地分类了，所有相应的对策都事先有了，只是等着市场自己去选择，去触及我们事先给定的开关。"
>
> ——课098（0902）

---

### 定义三：分段函数操作模型

定义：当 [操作指令严格按以下三分支且仅三分支执行] 时，称为符合分段函数操作模型：
- 分支1：当前走势状态 = 买点（第一/二/三类买点之一）→ 执行买入
- 分支2：当前走势状态 = 卖点（第一/二/三类卖点之一）→ 执行卖出
- 分支3：当前走势状态 ∈ [买点, 卖点] 区间 → 持有（前驱为买点则持股，前驱为卖点则持币）

不满足三分支完备覆盖（存在买卖点定义之外的第四种操作触发条件，如"感觉涨不动了"）则不构成分段函数操作模型。

> 原文引用："X=买点，买入；X=卖点，卖出；X属于买卖点之间，就持有，而这持有的种类，如果前面买点、卖点没出现，就是股票，反之就是钱。"
>
> ——课068（0619）

## 📐 定理

### 定理一：走势不可重复性定理

前提：[走势是市场所有参与者当下行为的合力结果] AND [参与者心理不可重复]
→ 结论：[任何历史走势模式不可能在完全相同条件下重复，因此基于"历史相同起点→相同结果"的机械预测不成立；操作必须响应当下走势而非套用历史模板]

> 原文引用："股票走势，归根结底，是参与者心理合力的痕迹，而心理，是不可重复的。"
>
> ——课086（0747）

---

### 定理二：走势自同构性定理（可操作性保证）

前提：[人的贪嗔痴疑慢（心理模式）在不同时间尺度上结构相同]
→ 结论：[走势在不同级别上显示出自相似性——分型、中枢、走势类型的结构在各级别均可复制；走势类型的基本形态在任何级别上都可判断，有绝对可操作性]

> 原文引用："本ID理论的哲学本质，就在于人的贪嗔痴疑慢所引发的自相似性以及由此引发走势级别的自组性这种类生命的现象。"
>
> ——课081（0695）

> ⚠️ 术语说明：此处"自相似性"为课081原文措辞（课84修订前）。课084（0718）已将该术语统一修正为"自同构性结构"，以避免与数学分形术语混淆。v2体系中应以"自同构性"为准，课081引用处保留原文措辞并标注。

---

### 定理三：走势合力当下构成定理

前提：[市场走势由所有参与者每笔交易行为的向量合成] AND [合力在每笔成交时当下完成]
→ 结论：[走势的具体点位、时间、幅度在成交前无法被任何单一分力（包括预测行为本身）精确确定；合力只在当下形成，走势是当下构成中的]

> 原文引用："市场的所有走势，都是当下合力构成。……无论是常量还是随着每笔成交变化的变量，合力都是当下构成的。"
>
> ——课068（0619）

---

### 定理四：预测即分力参与定理（市场版测不准原理）

前提：[预测行为本身是参与者行为之一] AND [参与者行为是走势的构成分力]
→ 结论：[预测行为本身改变被预测对象；任何关于具体点位的"非当下确定性预测"在理论上不成立；市场走势的基本形态可预测，具体点位是形态展开过程中的当下结果]

> 原文引用："站在哲学的角度，预测也是一个分力，就如同观察者本来就被假定在观察之中……正如同量子力学的测不准原理，任何关于预测的理论，其最大的原理就是测不准。"
>
> ——课068（0619）

---

### 定理五：形态可知／点位不可知定理（"不患"原理）

前提：[走势基本形态（上涨/下跌/盘整的结构类型）由参与者心理模式的自相似性决定] AND [走势具体点位由当下合力决定]
→ 结论：[形态是"不患"的（其类型边界条件可预先确定），点位是"不患之患"（形态展开的当下结果）；操作应把握形态，不执著于点位；把握"不患"后，"患"（点位）自然在当下的把握中得到]

> 原文引用："点位只是基本形态演化的一个结果，是当下中形成了，形态是'不患'的，点位是'不患'之'患'，只要把握了这'不患'，其'患'自然就在当下的把握中。"
>
> ——课052（0530）

---

### 定理六：走势时间不可逆定理

前提：[交易按时间排序，每笔交易有唯一时间戳] AND [已成交的走势不可撤销]
→ 结论：[过去的买卖点节点无法补做；操作必须实时响应当下走势，任何对已过去节点的补救操作均不符合当下性原则]

> 原文引用："交易是有时间性的，而这时间，不可逆。"
>
> ——课030（0459）

---

### 定理七：走势三支点定理（理论有效性根基）

前提：[走势具有不可重复性] AND [走势自同构结构具有绝对复制性] AND [缠论由纯逻辑推导构建]
→ 结论：[三支点共同保证：①不可重复性决定判断必须且必然建立在当下；②自同构复制性决定判断有绝对可操作性；③纯逻辑推导保证结论绝对有效性]

> 原文引用："走势的不可重复性，决定了一切的判断必须也必然是不可绝对预测的；自同构性结构的绝对复制性，决定了一切的判断都是可判断的，有着绝对的可操作性；理论的纯逻辑推导，就证明其结论的绝对有效性。"
>
> ——课084（0718）

---

### 定理八：技术分析当下直观定理

前提：[技术分析以价格走势为唯一输入] AND [走势是完全公开的当下第一手信息]
→ 结论：[技术分析的最终意义不是预测市场要干什么，而是直观市场正在干什么；所有技术分析错误都源于离开当下直观，用想象或情绪代替]

> 原文引用："技术分析的最终意义不是去预测市场要干什么，而是市场正在干什么，是一种当下的直观。在市场上所有的错误都是离开了这当下的直观，用想象、用情绪来代替。"
>
> ——课019（0420）

## 🔍 边界判断

### 区分：完全分类操作 vs 预测操作

遇到"操作决策"需判断其属于合法缠论操作还是预测操作 →
  先判断：操作触发条件是否依赖走势实际到达某可观测的边界状态？
    是 → 属于完全分类操作（合法缠论操作）
    否 → 判断：是否假设某特定情况先成立，再根据该假设推断后续操作？
      是 → 属于预测操作（违反当下性原则，缠论不采用）
      否 → 判断：决策是否有"只要满足X则必然出现Y"的排他性假设？
        是 → 仍为预测操作
        否 → 属于参考分析倾向，须等待走势自身选择后触发

---

### 区分：当下性使用 vs 破当下执

遇到"强调当下"与"破当下执"（课084）的张力 →
  判断：强调当下性的目的是什么？
    目的A：破除"预测点位"执念（绝大多数学习者所处阶段）→ 当下性有效，应使用
    目的B：当下性已内化，但"当下"本身变成另一种执念（口头禅/形式化/新的固化）→
      进入"破当下执"层次：
        判断：操作者是否已真正脱离点位预测执念？
          否 → 继续强化当下性训练，不可跳级进入"破当下执"
          是 → 参照📜经验判断「超逻辑直觉」行

> ⚠️ 注：课084明确指出"当下之所以被反复提出，就是因为有人企图预测的想法太过根深蒂固，只是一种破的手段"——当下性是对治预测执念的方法论，而非终极目标。

---

### 区分：技术分析方法的缠论有效性

遇到各类技术指标/方法（波浪理论、江恩、支撑阻力位等），判断是否为缠论接受 →
  先判断：该方法是否从固定先验前提出发（如"推动浪5波、调整浪3波"的神秘数字规律）？
    是 → 非当下性方法，缠论不采用；该类方法只能作为庄家制造骗线的工具
    否 → 判断：该方法最终是否落实到价格走势的当下可观测形态？
      是 → 可作为参考，但须在缠论框架内给出其成立的边界条件
      否 → 不构成缠论有效操作依据

---

### a+A+b+B+c 结构的当下级别判断

遇到 a+A+b+B+c 走势中需判断 B 的级别 →
  当 a+A+b 已出现，判断 b 段是否出现背驰：
    b 段出现背驰 → B（新中枢）即将形成，等待 B 确认
    b 段无背驰 →
      判断 b 段内部是否已形成次级别中枢：
        已形成次级别中枢 → b 段可能演化为更大级别，B 的级别将大于 A
          此时 a+A+b 合并重新分解为 a'，整体结构变为 a'+B
        未形成次级别中枢 → 继续等待，不可预判 B 的级别
  ⚠️ B 的级别在 B 完成前绝对不可确定，任何关于 B 级别的预先判断均违反当下性原则

## 💡 操作规则

### 操作规则一：分段函数买入执行规则

```
前提：[已确定当前操作级别] AND [走势结构已被该级别缠论框架解读] AND [当前未持仓]
触发：[当前走势到达该操作级别的买点（第一类买点：趋势背驰点 / 第二类买点：一买后次级别回调低点 / 第三类买点：中枢后离开再回抽不破ZG）]
操作：[识别买点类型] → [确认买点条件满足（参照各类买点充要条件）] → [执行买入] → [记录买入依据级别与买点类型] → [设止损参照（第一类：前低；第二类：一买低点；第三类：ZG）]
冲突优先级：当[同级别买点信号存在歧义时]，本规则让位于[降一级别重新识别结构后再判断]；当[更高级别未确认方向时]，本规则仍可执行但须控制仓位
```

---

### 操作规则二：分段函数卖出执行规则

```
前提：[当前持仓] AND [走势结构已被操作级别缠论框架解读]
触发：[当前走势到达该操作级别的卖点（第一类卖点：趋势背驰点 / 第二类卖点：一卖后次级别反弹高点 / 第三类卖点：中枢后离开再回抽不破ZD）]
操作：[识别卖点类型] → [确认卖点条件满足] → [执行卖出] → [卖出后不再预设该股回购条件，除非新买点出现]
冲突优先级：本规则优先于[持股惯性/对股票的情感依恋]；卖点出现在上涨途中，宁愿卖早不可卖晚；若已砍在地板价则必然不是在卖点上
```

---

### 操作规则三：a+A+b+B+c 当下跟踪操作规则

```
前提：[已识别某级别走势处于 a+A+b 结构中] AND [A 为已确认同级别中枢（至少三个次级别走势类型价格区间重叠，ZD < ZG）] AND [已在 A 内或 a 段买点入场]
触发：[b 段（次级别走势）出现背驰信号]
操作：[确认 b 段背驰成立] → [b 段背驰点附近减仓或出仓] → [等待次级别回拉：判断低点是否跌破 A 中枢的 ZG] → [若不跌破 ZG（第三类买点条件满足）→ 回补建仓] → [等待 c 段展开并按 c 段背驰点再次操作]
冲突优先级：当[b 段无背驰但已有次级别中枢可能演化为更大 B]时，本规则暂停等待，不强行判断 B 的级别；级别判断优先于操作触发
```

---

### 操作规则四：完全分类触发操作规则

```
前提：[当前走势已完成某结构节点（如背驰点/买卖点）] AND [后续存在多种可能演化方向]
触发：[结构节点完成后立即执行（无须外部触发）]
操作：[穷举后续所有可能走势状态（例：背驰后→回拉是否跌回中枢/是否形成新中枢/是否直接下破）] → [为每种状态预设对应操作指令（状态X→操作P；状态Y→操作Q）] → [进入等待模式：等待走势实际到达某状态的边界条件] → [边界条件触达时执行预设操作]
冲突优先级：本规则优先于任何基于单一假设的预测操作；分类必须穷举，不允许遗漏任何可能状态
```

---

### 操作规则五：操作级别锁定与跨级别冲突规则

```
前提：[操作者已选定并锁定操作级别（如年线/日线/30分钟/5分钟）]
触发：[走势到达该操作级别的买卖点]
操作：[在该级别买点买入] → [持有至该级别卖点出现] → [卖出]
冲突优先级：当[低级别出现买卖点信号但高级别无信号时]，低级别信号不触发高级别持仓的操作指令；不同级别操作彼此独立，级别是操作的绝对框架，不可混用；年线级别操作买入后可能数十年无卖点，须按级别框架执行持仓
```

## 📜 经验判断

> ⚠️ **原文未给出充要条件**，以下为原文经验性指引，AI 使用时须视为参考倾向而非确定规则。

| 场景 | 判断倾向（原文指引） | 边界（不适用情形） |
|------|-------------------|--------------------|
| 场景：[操作者反复对走势感到不满，希望走势按预想方向发展] | 判断倾向：[这是预测性思维残留，应转换为"依据市场的选择及时反应"而非"为市场的选择而烦恼"] | 边界：[若不满来自对自身操作失误的复盘反思，属于合理学习，非当下性问题] |
| 场景：[操作者已理解理论但操作仍患得患失、瞻前顾后] | 判断倾向：[应优先通过逻辑推导彻底搞清楚理论的严格性（如几何证明的确定性），建立信心再操作；理论探讨的心理层面意义与分析指导意义同等重要] | 边界：[若已充分理解理论但仍有操作偏差，多为习惯性多空头问题，强化理论学习已非优先项] |
| 场景：[操作者存在习惯性多头（买早卖晚）或习惯性空头（买晚卖早）偏差] | 判断倾向：[承认习惯性力量的存在，在信号出现时主动对抗惯性偏差；这种习惯性力量的改变不可能是一天两天的事] | 边界：[不适用于新手阶段；新手应先机械遵守买卖点原则，习惯纠偏是进阶问题] |
| 场景：[操作者看图时被"我"和"股票"的情绪执念牵引，看到的是自己的贪婪恐惧而非走势本身] | 判断倾向：[目标状态为"无我无股票，只有走势图形"；看图操作时唯一对象是图；市场是当下发生的，无人能替你反应] | 边界：[此为长期修炼目标，短期强制压制情绪可能导致反效果；应通过机械化执行逐步训练，不可急于求成] |
| 场景：[当下性原则已内化，操作者希望突破当下性本身的限制] | 判断倾向：[破当下执：当下性是破除预测执念的手段而非终点；"过去心不可得，现在心不可得，未来心不可得"，进入超逻辑直觉层次] | 边界：[仅适用于已能稳定做到无预测操作的熟练操作者；初学者不应跳过当下性直接进入"破当下执"，否则只是口头禅] |
| 场景：[操作者寻求1秒钟看懂日线图大势的直觉判断能力] | 判断倾向：[此为理论分析的最终归宿（"心物相通"）；前期理论学习是"门外的热身"，最终须将理论内化为直觉；到达此层次时中枢等概念可以退隐，只剩当下的直接感应] | 边界：[必须先经过严格理论训练，不可跳过；跳过理论直接追求"直觉"与缠论方法论相悖] |
| 场景：[操作者按同级别分解节奏进行机械化操作训练] | 判断倾向：[持续按走势类型分解的韵律操作，人体生物节律会慢慢感应走势节奏；当该操作图形出现时，生理上会有自然感应] | 边界：[生理感应为进阶现象描述，不可反向作为操作依据（即不能因"今天没有生理感应"而放弃合法买卖点）；须以理论正确操作为前提] |
| 场景：[走势呈现多义性（同一走势可有多种合法分解），操作者感到困惑] | 判断倾向：[多义性不是模糊性，而是多角度让市场自行画地为牢；在超逻辑直觉未形成前，多义性的完全分类分析依然必要；形成充分直觉后多义性自然消解] | 边界：[走势多义性的形态学处理属于P4同级别分解范畴；此处仅保留其当下性哲学维度（多义分析是当下完全分类的体现）] |
