# P3-10-中枢震荡与延伸

> **文件用途说明**：本文件是面向 AI 系统的缠论知识点文件，内容须自洽、无歧义、可机器解析。
> 所有定义优先使用充要条件格式，经验性判断须单独标注并与精确定义区分。

## 概述

中枢震荡与延伸是缠论 P3-中枢理论的核心动态框架，描述走势中枢建立后价格持续运动的全部可能模式。
本章引入 Z走势段 与 GG/G/D/DD 四指标，通过中枢中心定理一给出中枢延伸的充要条件；建立进场三类框架、高抛低吸差价及中枢移动满仓的完整中枢震荡操作体系；并以震荡中轴 Z/Zn 序列作为变盘预警工具。
原文课18至课92间渐进展开，核心定理以课20为基准，操作体系以课49为基准，监视器工具以课92为基准，无版本冲突。

## 📖 定义

### 定义一：Z走势段（Z-Segment）

定义：当且仅当某次级别走势类型同时满足以下两个条件时，称为该走势中枢的 **Z走势段**（按时间顺序记为 Z₁, Z₂, Z₃…Zₙ）：
  - 条件一：该次级别走势类型发生在某已确立走势中枢的形成或延伸过程中；
  - 条件二：该次级别走势类型的运动方向与该中枢的形成方向一致。
不满足任一条件，则该次级别走势类型不构成该中枢的Z走势段。各 Zₙ 的高点记为 gₙ，低点记为 dₙ。

补充：
- 回升形成的中枢，Z走势段为向上次级别走势（A、B、C 三段中 A/C 段为 Z段）
- 回调形成的中枢，Z走势段为向下次级别走势
- 中枢区间可由前两个 Z段确定简化公式：ZG = min(g₁, g₂)，ZD = max(d₁, d₂)，即 [max(a₂,c₂), min(a₁,c₁)]

> "以后都把这些与中枢方向一致的次级别走势类型称为Z走势段，按中枢中的时间顺序，分别记为Zn等，而相应的高、低点分别记为gn、dn。"
>
> ——课20（0424）

---

### 定义二：GG / G / D / DD 四指标

定义：对已确立走势中枢，令 n 遍历所有 Z走势段，定义如下四个全局指标：

| 指标 | 公式 | 含义 |
|------|------|------|
| **GG** | max(gₙ) | 所有 Z段高点的最大值（中枢历史最高极值）|
| **G**  | min(gₙ) | 所有 Z段高点的最小值 |
| **D**  | max(dₙ) | 所有 Z段低点的最大值 |
| **DD** | min(dₙ) | 所有 Z段低点的最小值（中枢历史最低极值）|

补充：ZG/ZD 由中枢形成时的 Z₁/Z₂ 固定，之后不随延伸更新；GG/DD 随每个新 Zₙ 动态更新，恒有 GG ≥ ZG ≥ ZD ≥ DD。

> "定义四个指标，GG=max(gn)，G=min(gn)，D=max(dn)，DD=min(dn)，n遍历中枢中所有Zn。特别地，再定义ZG=min(g1、g2)，ZD=max(d1、d2)，显然，[ZD，ZG]就是缠中说禅走势中枢的区间。"
>
> ——课20（0424）

---

### 定义三：中枢延伸（Pivot Extension）

定义：当走势中枢已形成，且后续每一个 Z走势段 Zₙ 的区间 [dₙ, gₙ] 均与中枢区间 [ZD, ZG] 存在非空重叠时（即 dₙ ≤ ZG 且 gₙ ≥ ZD），该中枢处于**延伸**状态；若存在任意 Zₙ 使得 dₙ > ZG 或 gₙ < ZD，则中枢延伸终止，必然产生更高级别走势中枢或趋势延续。

充要条件（等价于中枢中心定理一）：
```
中枢延伸 ⟺ ∀Zₙ: [dₙ, gₙ] ∩ [ZD, ZG] ≠ ∅
         ⟺ ∀Zₙ: dₙ ≤ ZG  AND  gₙ ≥ ZD
```

终止后果：
- ∃Zₙ: dₙ > ZG → 向上脱离中枢 → 必然产生更高级别中枢或上涨趋势延续
- ∃Zₙ: gₙ < ZD → 向下脱离中枢 → 必然产生更高级别中枢或下跌趋势延续

> "走势中枢的延伸等价于任意区间[dn，gn]与[ZD，ZG]有重叠。换言之，若有Zn，使得dn>ZG或gn<ZD，则必然产生高级别的走势中枢或趋势及延续。"
>
> ——课20（0424）

---

### 定义四：走势类型延伸（Trend-Type Extension）

定义：当某级别走势类型已满足最小完成条件（盘整：已有一个中枢；趋势：已有两个同向中枢），但尚未产生新的走势中枢，而是继续在原结构基础上增加次级别走势类型时，称为**走势类型延伸**。
判断走势类型延伸是否结束，当且仅当是否产生新的走势中枢：
- 未产生新中枢 → 走势类型延伸继续
- 产生新中枢 → 走势类型完成，下一走势类型开始

> "盘整就可以随时完成……但这可以不结束，可以不断延伸下去，不断围绕这缠中说禅中枢上上下下地延伸下去直到无穷都是可以的。"
>
> ——课18（0419）

> "走势类型延伸是否结束的判断关键就在于是否产生新的'缠中说禅走势中枢'。"
>
> ——课18（0419）

---

### 定义五：震荡中轴 Z 与 Zₙ 序列（Oscillation Axis）

定义：对已确立走势中枢 [ZD, ZG]（记为区间 [A, B]），定义：
- **震荡中轴 Z** = (ZD + ZG) / 2 = (A + B) / 2（中枢区间中点，为不变参照基线）
- **次级震荡中点 Zₙ** = (dₙ + gₙ) / 2（第 n 次 Z走势段区间的中点）

性质：A、Z、B 三线等距（Z - A = B - Z = (B - A) / 2）。Zₙ 序列连成曲线，构成监视中枢震荡强弱的技术指标。Zₙ 的数量一般不超过 9 个（原文经验性描述："Zn的数量不会过于庞大，不会超过9个数据，超过了，次级别就要升级了"——课92；此为经验上限而非严格充要条件，实际扩张的判定以 Zₙ 是否突破 A/B 边界为准，见定理七）。

> "一个中枢确立后，中枢区间的一半位置，称为震荡中轴Z。而每一个次级震荡区间的一半位置，依次用Zn表示……A、Z、B这三条直线刚好是等距的，Zn的波动连成曲线，构成一个监视中枢震荡的技术指标。"
>
> ——课92（0844）

### 定义六：中阴阶段（Shadow Transition Phase）

定义：当某级别走势类型已因背驰（第一类买卖点）宣告结束，新走势类型尚未通过自身次级别结构独立确立之前，该过渡阶段称为**中阴阶段**。

形态充要特征（原文明确）：无一例外表现为围绕前一走势类型某局部区域构成的中枢震荡（即某级别的盘整形态）。
- V 型反转也属于中阴阶段，仅震荡幅度更深
<!-- ⚠️ 注意：「中阴阶段表现为盘整」描述截取该段的形态，不代表所在的更大级别走势类型必然是盘整 -->

> "中阴阶段，无一例外，都是表现为不同级别的盘整……也就是围绕前一走势的某一部分所构成的中枢震荡，即使是所谓的V型反转，也一样，只是震荡的区域回得更深而已。"
>
> ——课88（0767）

## 📐 定理

### 定理一：中枢中心定理一（延伸充要条件）

前提：[走势中枢已确立，区间为 [ZD, ZG]；后续 Z走势段序列 Zₙ 各区间为 [dₙ, gₙ]]
→ 结论：中枢延伸 **当且仅当** ∀Zₙ: [dₙ, gₙ] ∩ [ZD, ZG] ≠ ∅；若存在 Zₙ 使 dₙ > ZG 或 gₙ < ZD，则必然产生更高级别走势中枢或趋势延续。

> "走势中枢的延伸等价于任意区间[dn，gn]与[ZD，ZG]有重叠。"
>
> ——课20（0424）

---

### 定理二：中枢中心定理二（前后中枢位置关系三分类）

前提：[前后两个同级别走势中枢已确立；前中枢 GG/DD 为前GG/前DD，后中枢 ZG/ZD/GG/DD 为后ZG/后ZD/后GG/后DD]
→ 结论（三类完备分类）：
1. **后GG < 前DD** ⟺ 下跌及其延续
2. **后DD > 前GG** ⟺ 上涨及其延续
3. (后ZG < 前ZD 且 后GG ≥ 前DD) OR (后ZD > 前ZG 且 后DD ≤ 前GG) ⟺ 形成更高级别走势中枢

> "后GG < 前DD等价于下跌及其延续；后DD > 前GG等价于上涨及其延续。后ZG<前ZD且后GG>=前DD，或后ZD>前ZG且后DD=<前GG，则等价于形成高级别的走势中枢。"
>
> ——课20（0424）

---

### 定理三：走势级别延续定理一

前提：[某级别走势进行中，且尚未形成更高级别走势中枢]
→ 结论：该级别走势类型将延续——只能是含该级别中枢的盘整或趋势的延续；更大级别的结构转变在此阶段不可能发生。

> "在更大级别缠中说禅走势中枢产生前，该级别走势类型将延续。"
>
> ——课20（0424）

---

### 定理四：走势级别延续定理二

前提：[同级别相邻两个走势中枢已存在]
→ 结论：更高级别走势中枢形成 **当且仅当** 围绕这两个中枢产生的波动区间之间存在重叠。

> "更大级别缠中说禅走势中枢产生，当且仅当围绕连续两个同级别缠中说禅走势中枢产生的波动区间产生重叠。"
>
> ——课20（0424）

---

### 定理五：背驰发生位置定理

前提：[某级别趋势存在（至少两个同向中枢，中枢间无重叠）]
→ 结论：
- 趋势背驰不发生在第一个中枢之后
- 真正的趋势背驰至少在第二个中枢之后
- 在第一个中枢后出现的力度衰减仅构成**盘整背驰**：技术含义为企图脱离中枢的运动因力度不足被回拉，并非真正的趋势背驰

> "趋势，一定有至少两个同级别中枢，对于背驰来说，肯定不会发生在第一个中枢之后，肯定是至少是第二个中枢之后。"
>
> ——课27（0449）

---

### 定理六：中枢震荡终结定理

前提：[某级别中枢处于延伸（震荡）状态]
→ 结论：该中枢震荡最终必然以某级别的**第三类买卖点**结束——不存在永久震荡；若中枢不断延伸升级为更高级别中枢，则按更高级别的第三类买卖点处理，如此类推，总要面临某一级别的第三类买卖点终结该震荡。

> "这5分钟中枢成立后，就必然100%面临一个破坏的问题，也就是一个延伸或着第三买卖点的问题……总要面临某一个级别的第三买卖点去结束这个中枢震荡。"
>
> ——课89（0782）

---

### 定理七：Zₙ 突破必然性定理

前提：[某中枢处于震荡延伸中，Zₙ 序列存在]
→ 结论：Zₙ **必然**最终突破 A（ZD）或 B（ZG）——若 Zₙ 永不突破，则第三类买卖点将永不出现，与「走势终完美」公理矛盾。
注意：Zₙ 突破 A/B 是第三类买卖点出现的必要条件而非充要条件，可能出现多次假突破后才最终确认。

> "存在着一种必然的关系，就是最终，Zn肯定要超越A或B……如果不这样，就永远不会出现第三类买卖点了，这显然是不可能的。"
>
> ——课92（0844）

---

### 定理八：走势本质定理

前提：[任意级别、任意走势图形]
→ 结论：任何走势的运转模式，本质是**不同级别的中枢震荡与中枢移动的组合**。震荡（中枢延伸）提供差价机会，移动（中枢向上/下推移）提供趋势获利机会；走势图形的差异来自各股的频率、幅度、形态复杂度，而非模式本质不同。

> "股票的运转模式，归根结底，就是不同级别的中枢震荡与移动的组合最终构成相应的前戏-高潮模式……这也就是为什么，依据同一模式展开的走势，却呈现千差万别的最终图形。"
>
> ——课55（0538）

---

### 定理九：第一利润最大定理

前提：[固定交易品种，确定操作级别 L]
→ 结论：以下缠中说禅操作模式的利润率最大：①中枢震荡期高抛低吸做差价（降低持股成本，不丢失筹码）；②中枢第三类买点出现后满仓持有（跟随中枢向上移动）；③中枢完成的向上移动出现背驰后全部抛出；全程不参与 L 级别以上的下跌和超过 L 级别的盘整。

> "只有围绕操作级别中枢震荡的差价才是最安全的，因为肯定能做出来，而且绝对不会丢失筹码……中枢向上移动时，就应该满仓，这才是最正确的仓位。"
>
> ——课49（0515）

## 🔍 边界判断

### 中枢震荡 vs 中枢延伸 vs 中枢扩张的判断树

遇到中枢附近价格反复运动，需确定所处状态 →

```
步骤1：检查所有已完成 Zₙ 区间是否均与 [ZD, ZG] 有重叠
  dₙ ≤ ZG 且 gₙ ≥ ZD  对所有 n 成立
  → 中枢处于【延伸（震荡）】状态
     → 步骤2：检查 Zₙ 数量
          ≤ 9 个 Z段（额外延伸 ≤ 5 段）
          → 正常延伸，级别不变
          ≥ 9 个 Z段（额外延伸 ≥ 6 段）
          → 经验上大概率已触发中枢级别扩张，重识别为更高级别中枢（经验性阈值，非严格充要条件；严格判定见步骤1中 Zₙ 是否突破 [ZD, ZG] 边界）

  存在 dₙ > ZG（某 Zₙ 完全在 ZG 之上）
  → 中枢延伸终止，向上脱离
     → 步骤3：判断其后次级别回调
          回调低点不跌破 ZG → 构成【第三类买点】，进入新上涨
          回调低点跌破 ZG 返回中枢 → 判断与前中枢波动是否重叠：
            有重叠 → 形成【更高级别中枢】
            无重叠 → 继续观察

  存在 gₙ < ZD（某 Zₙ 完全在 ZD 之下）
  → 中枢延伸终止，向下脱离（逻辑完全对称，判断第三类卖点或中枢扩张）
```

**概念关系澄清**（避免混用）：
- **中枢震荡**（口语）= 价格围绕中枢反复运动的行为性描述
- **中枢延伸**（技术术语）= 所有 Zₙ 满足与 [ZD,ZG] 重叠的精确定义
- AI 判断时以「中枢延伸」的充要条件为准，「中枢震荡」仅作行为描述使用

### 单价位重叠中枢的有效性

遇到两段次级别走势仅有单一价位重叠（ZD = ZG）→
  判断是否满足 ZD ≤ ZG：
    ZD = ZG（恰为单点）→ **有效中枢**，不可忽略；原文明确单价位重叠有时意义更大（常为主力资金操作痕迹）
    ZD > ZG（无任何重叠）→ 不构成中枢

---

### 走势多义性处理

遇到同一段走势可解释为不同级别中枢结构 →
  检查是否存在向下倾斜的「第三段」——若疑似中枢内部的第三 Z段实为向下延续（非独立反向走势）→ 该段不算 Z段，不构成中枢，整体为更大走势段的组成部分。
  若各段确实独立且方向交替 → 核实区间重叠，满足则构成中枢；多种合法分解均符合走势结合律，选操作级别最匹配的分解角度。

> "B段，看似要形成3个1分钟的中枢，但由于每一个的第三段其实都是向下倾斜下去的，其实都是第二段向下的一部分，不能算是形成中枢。"
>
> ——课56（0541）

---

### 盘整背驰 vs 趋势背驰的判断

遇到中枢附近出现力度衰减迹象 →
  判断当前走势中的中枢数量：
    仅一个中枢（盘整走势）→ 只能是**盘整背驰**，技术含义为企图脱离中枢被回拉，不等同于走势类型终结
    至少两个同向中枢（趋势走势）且背驰发生在第二个及以后的中枢之后 →**趋势背驰**，是走势类型终结的信号
    至少两个同向中枢但背驰迹象出现在第一个中枢之后 → 仍为**盘整背驰**，不构成趋势背驰

## 💡 操作规则

### 操作规则一：进场三类框架（当下位置分类）

前提：[已确定操作级别 L（如 30 分钟）] AND [准备入市或换股]
触发：[找到当下之前最后一个 L 级别走势中枢，确认其区间 [ZD, ZG]]
操作：
```
判断当下价格在中枢中/上/下（三类之一）：

【类一：当下在中枢 [ZD, ZG] 之中】
→ 最优操作：不操作，等待演化为类二或类三
→ 次优操作：若具备次级别第二类买点判断能力，可小仓参与

【类二：当下在中枢之下】
→ 子类 2-1（未出现第三类卖点，中枢震荡依旧）：
   → 用 MACD 比较当下向下段与前次同向段力度
   → 出现盘整背驰迹象 → 按区间套法逐级精确确认买点入场
→ 子类 2-2（已出现第三类卖点，中枢已完成）：
   → 分析包含该卖点的次级别走势完成，用背驰法找当前级别买点
   → 或等待新中枢形成后再介入（避免参与超过操作级别的盘整）

【类三：当下在中枢之上】
→ 子类 3-1（未出现第三类买点）：无买点，不操作，等待
→ 子类 3-2（已出现第三类买点）：
   → 距买点形成位置不远可介入
   → 若次级别走势已完成并出现盘整顶背驰 → 等待该盘整结束
```
冲突优先级：操作级别未确定时本规则不可执行；L 级别以上的下跌和超过 L 级别的盘整一律不参与

> "不操作是最好的操作，等待其演化成第二、三类……只把握你自己当下技术水平能把握的机会，这才是最重要的。"
>
> ——课49（0515）

### 操作规则二：中枢震荡高抛低吸差价

前提：[操作级别中枢处于延伸状态（未出现第三类买卖点）] AND [已持仓或准备建仓]
触发：[价格在中枢区间 [ZD, ZG] 附近运动，次级别出现顶/底背驰信号]
操作：
```
价格向上离开中枢（在 ZG 之上）且次级别出现顶背驰（向上离开力度弱于前次）
→ 减仓 / 抛出

价格向下离开中枢（在 ZD 之下）且次级别出现底背驰（向下离开力度弱于前次）
→ 加仓 / 回补

理想全仓操作：中枢上方全部抛出，中枢下方全数接回（需极高技术精度，不熟练者可减仓操作）
```
冲突优先级：
- 前提消失（出现第三类卖点信号）→ **立即停止回补，规则失效**，转为操作规则三
- 本规则仅在「中枢震荡依旧」条件成立时有效

> "在围绕中枢差价时，在中枢上方仓位减少，在中枢下方仓位增加，注意，前提是中枢震荡依旧，一旦出现第三类卖点，就不能回补了。"
>
> ——课49（0515）

### 操作规则三：中枢移动满仓持有

前提：[操作级别中枢第三类买点已确认（次级别回调低点不跌破 ZG）] AND [价格开始向上移动]
触发：[第三类买点形成，走势进入中枢完成后的向上移动阶段]
操作：
```
→ 立即转为满仓持有，停止一切差价操作
→ 持有至新中枢形成后，继续用操作规则二处理新中枢震荡
→ 中枢完成的向上移动出现背驰 → 全部抛出，完成本轮操作，等待下一买点
```
冲突优先级：本规则**优先于**操作规则二——中枢移动阶段禁止做差价，否则丢失筹码，牺牲最大利润段

> "那种中枢完成后的向上移动时的差价是不能做的，中枢向上移动时，就应该满仓，这才是最正确的仓位。"
>
> ——课49（0515）

### 操作规则四：Zₙ 监视器信号处理

前提：[中枢 [ZD, ZG]（即 [A, B]）已确立，Z = (A + B) / 2 已计算]
触发：[每次次级别震荡完成，计算 Zₙ = (dₙ + gₙ) / 2]
操作：
```
Zₙ > Z（偏强）→ 维持仓位，倾向中枢上沿方向操作
Zₙ < Z（偏弱）→ 注意下行风险，减少仓位

Zₙ 突破 B（超过 ZG）：
  → 变盘警示，判断是否构成第三类买点：
      次级别回调低点不破 ZG → 第三类买点确认 → 转入操作规则三（满仓持有）
      价格返回中枢 → 大概率构成中枢级别扩张，重新识别更高级别中枢

Zₙ 突破 A（低于 ZD）：
  → 变盘警示（逻辑对称），判断第三类卖点或中枢扩张

Zₙ 缓慢递增但长期无法突破 B → 警惕上升楔形诱多，准备反向应对
Zₙ 缓慢递减但长期无法突破 A → 警惕下降楔形诱空
```
冲突优先级：Zₙ 为辅助预警工具，不能替代第三类买卖点的走势结构确认；Zₙ 多次假突破均合法，以结构确认为最终依据

> "Zn在Z之上，证明这个震荡是偏强的，反之偏弱……一旦有这类似的超越，就是一个很大的提醒，也就是这震荡面临变盘了。"
>
> ——课92（0844）

### 操作规则五：中枢震荡多义分解背驰操作

前提：[同一段走势可按多种方式分解出不同中枢划分（走势多义性）]
触发：[在某种合法分解下，可将当前走势识别为某中枢的震荡段]
操作：
```
→ 选择能合理说明当前力度背驰判断的合法分解方式
→ 将当前疑似背驰的段设为比较段，与前次同方向离开段比较 MACD 力度
→ 出现盘整背驰信号 → 据此做差价（按操作规则二处理）
```
冲突优先级：所有分解必须满足走势结合律，不可任意拼凑；多种合法分解可同时存在，提供多角度风险控制

> "这分法，就使得23、67成为这中枢的一个震荡，从而可以用力度的方法来发现背驰。"
>
> ——课57（0542）

## 📜 经验判断

> ⚠️ **原文未给出充要条件**，以下为原文经验性指引，AI 使用时须视为参考倾向而非确定规则。

| 场景 | 判断倾向（原文指引） | 边界（不适用情形） |
|------|-------------------|--------------------|
| 场景：一个趋势（下跌/上涨）中出现多个中枢，判断是否接近终结 | 判断倾向：通常最多 3～4 个中枢；日线以上级别第二个中枢后背驰概率约 90%；超过 4 个中枢极为罕见；第三类卖点下来的次级别背驰通常只制造一个下跌中枢，不构成真正趋势背驰 | 边界：超大级别走势或特殊行情（如连续涨跌停）可能突破一般规律；理论上趋势可无限延伸，不可机械套用数量上限 |
| 场景：中枢震荡中后续离开力度大于前次（扩张型震荡）| 判断倾向：扩张型震荡也属合法中枢延伸，不必然破坏中枢；但需结合 Zₙ 是否超越边界加强变盘监控 | 边界：若 Zₙ 超越边界后次级别维持在边界外，则扩张型后可能紧接第三类买卖点出现 |
| 场景：Zₙ 超越边界（A 或 B）但未构成第三类买卖点，判断后续 | 判断倾向：大概率构成中枢震荡级别扩张，应重新按更高级别中枢分析 | 边界：无 100% 绝对性；仍需其后走势确认；极少数情况价格多次假突破后才最终确认第三类点 |
| 场景：底部区域进行大级别建仓，走势极为复杂 | 判断倾向：底部区域可进行最复杂的中枢延伸与扩展，目的是累积低成本筹码；有时表现为斜率极小的通道式上升（充满激烈震荡），应借震荡降低成本而非离场 | 边界：若出现明确的大级别第三类卖点信号，则底部建仓假设失效，须重新判断 |
| 场景：中阴阶段形态识别（截取该段）| 判断倾向：无一例外表现为不同级别盘整（围绕前一走势某局部构成的中枢震荡）；V 型反转只是震荡区域更深的中阴阶段 | 边界：中阴阶段的形态判断仅适用于截取的该段；不代表所在的更大级别走势类型必然是盘整 |
| 场景：Zₙ 缓慢递增但长期无力突破 B，判断走势性质 | 判断倾向：此形态（上升楔形）蕴含突然向下变盘的风险，为诱多形态，应逐步减仓而非追涨 | 边界：若次级别出现明确背驰反转信号支持向上，则诱多判断失效，以结构判断为准 |
