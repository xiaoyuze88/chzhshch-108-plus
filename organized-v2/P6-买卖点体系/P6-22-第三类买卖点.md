# P6-22-第三类买卖点

> **文件用途说明**：本文件是面向 AI 系统的缠论知识点文件，内容须自洽、无歧义、可机器解析。
> 所有定义优先使用充要条件格式，经验性判断须单独标注并与精确定义区分。

## 概述

第三类买卖点是缠论买卖点体系的第三类操作信号，专门标志走势中枢结束（中枢上移形成趋势，或中枢扩张升维）的时机。
本章定义第三类买点/卖点充要条件，建立"三买由ZG不跌破确认"定理、"必须针对最近中枢"边界原则及"至少5段次级别"结构要求，为趋势开启前的最终确认信号提供完备的操作框架。
原文核心定理在课20（0424）由中枢中心定理一推导给出；课58（0545）补充"针对最近中枢"原则；课79（0682）给出面向散户的完整三买买卖法六步骤。

## 📖 定义

### 定义一：第三类买点

定义：当且仅当以下三个条件同时满足时，某价格点位构成某级别的**第三类买点**：
- 条件一：该级别走势中枢已形成，中枢区间 [ZD, ZG] 已确定（ZD = max(d1,d2)，ZG = min(g1,g2)）；
- 条件二：一个次级别走势类型完整地向上离开该中枢（该次级别走势类型整体高于 ZG，即作为离开段的次级别走势类型已完成）；
- 条件三：随后以另一个次级别走势类型回试该中枢，其低点不跌破 ZG（低点 ≥ ZG），且该回试为离开后的**第一次**次级别回试。

不满足任一条件则不构成第三类买点，包括：中枢未形成、离开段次级别走势不完整、回试低点跌破 ZG（低点 < ZG）、该次离开后已发生过第一次次级别回试。第三类买点的位置为回试次级别走势类型结束时的最低点。

> 原文引用："由定理一，可以得到第三类买卖点定理：一个次级别走势类型向上离开缠中说禅走势中枢，然后以一个次级别走势类型回试，其低点不跌破ZG，则构成第三类买点；一个次级别走势类型向下离开缠中说禅走势中枢，然后以一个次级别走势类型回抽，其高点不升破ZD，则构成第三类卖点。"
>
> ——课20（0424）

---

### 定义二：第三类卖点

定义：当且仅当以下三个条件同时满足时，某价格点位构成某级别的**第三类卖点**：
- 条件一：该级别走势中枢已形成，中枢区间 [ZD, ZG] 已确定；
- 条件二：一个次级别走势类型完整地向下离开该中枢（整体低于 ZD，已完成）；
- 条件三：随后以另一个次级别走势类型回抽该中枢，其高点不升破 ZD（高点 ≤ ZD），且为离开后的第一次次级别回抽。

不满足任一条件则不构成第三类卖点。第三类卖点位置为回抽次级别走势类型结束时的最高点。

来源：课20（0424）——同定义一原文一并给出，定义对称成立。

---

### 定义三：三买100%获利性质

定义：当且仅当某买点满足第三类买点充要条件（定义一三个条件全部满足）时，该买点具备**100%获利性质**：无论后续走势形成新的同级别上涨趋势（中枢新生），还是当前中枢扩展为更大级别中枢（中枢扩张），在第三类买点处介入后均必然出现高于买入价的价位，操作必然获利。

若第三类买点条件未满足（如低点跌破 ZG），则此性质失效，不可使用。

来源：课53（0534）——"在本ID的理论里……最晚就是在第三类买点介入，这是最安全的，100%获利。"

## 📐 定理

### 定理一：三买由中枢中心定理一推导（ZG不跌破定理）

前提：[某级别走势中枢已确定，存在 Zn 使得 dn > ZG（此处使用中枢中心定理一买点方向分支：dn > ZG，中枢延伸已终结）] AND [随后的次级别回试走势完成，其低点 ≥ ZG]
→ 结论：[第三类买点成立；该回试低点等价于中枢最终结束的确认，此后走势必然进入趋势（中枢新生）或更大级别中枢（中枢扩张）]

逻辑链：中枢中心定理一（dn > ZG 或 gn < ZD → 必然产生高级别中枢或趋势；此处仅用买点方向分支 dn > ZG）→ 回试不破 ZG → 延伸确实结束 → 第三类买点确立 → 100%获利性质成立。

来源：课20（0424），由中枢中心定理一推导，原文明确标注"由定理一，可以得到第三类买卖点定理"。

---

### 定理二：三买后两种情况（完备性）

前提：[第三类买点已出现（定义一三个充要条件全满足）]
→ 结论：后续走势**必然且只能**属于以下两种情况，无第三种可能：
- 情况A（中枢新生）：形成新的同级别走势中枢，走势进入上涨趋势；
- 情况B（中枢扩张）：当前中枢扩展为更大级别走势中枢。

无论情况 A 还是 B，都必然出现高于第三类买点的价位，100%获利性质由此成立。

来源：课21（0428）——"第三类买点是中枢扩张或新生产生的。中枢扩张导致一个更大级别的中枢，而中枢新生，就形成一个上涨的趋势，这就是第三类买点后必然出现的两种情况。"

---

### 定理三：三买必须针对最近中枢（最近中枢原则）

前提：[走势中存在多个历史中枢，需判断第三类买卖点针对哪个中枢]
→ 结论：第三类买卖点**当且仅当**针对该级别**时间上最近**的那个中枢判断 ZG/ZD；以更早历史中枢的 ZG/ZD 为判断基准所识别的信号不构成有效第三类买卖点，无论其价格关系如何。

来源：课58（0545）——"第三类买卖点必须是次级别离开，次级别反抽，而且是针对该级别中最近那个中枢。"

---

### 定理四：三买最低结构要求（5段次级别）

前提：[判断某价格点位是否满足第三类买卖点充要条件]
→ 结论：第三类买卖点的构成**至少**需要5段次级别走势类型：

| 段序 | 方向 | 作用 |
|------|------|------|
| 段1、段2、段3 | 三段次级别，存在价格区间重叠 | 构成该级别走势中枢 |
| 段4 | 次级别走势，完整离开中枢 | 离开段（dn > ZG 或 gn < ZD） |
| 段5 | 次级别走势，回试，第一次 | 构成第三类买卖点（低点 ≥ ZG 或高点 ≤ ZD） |

不足5段次级别走势则结构不完整，不构成有效三买卖点。

来源：课61（0558）——"一个第三类买卖点，至少需要有5段次级别的，前三段构成中枢，第四段离开中枢，第5段构成第三类买卖点。"

---

### 定理五：B中枢须有三买才算完成

前提：[分析 a+A+b+B+c 结构，判断 B 中枢是否结束]
→ 结论：中枢 B **当且仅当**出现该级别的第三类买点时才算完成（中枢结束）；若未出现第三类买点，则后续的 b、c 段之间的力度比较只能按盘整背驰处理，不能认定为趋势背驰，B 中枢仍处于延伸或震荡状态。

来源：课51（0521）——"B当然要出现第三类买点，否则B就没结束，都是围绕B的震荡，用盘整背驰就足以。"

---

### 定理六：三买的级别精确定位

前提：[以日线级别中枢为例，需精确定位三买买点]
→ 结论：若操作级别为日线，则判断三买的次级别为30分钟级别；次级别走势的完成还需要再次级别（5分钟）呈现3段走势类型才能确认。精确买点出现在再次级别（5分钟）三段走势完成时的低点，而非任意价格回落处。

来源：课20（0424）评论——"对日线中枢来说，次级别是30分钟，而次级别的完成，需要再次级别，也就是5分钟图上呈现3段走势类型。"

## 🔍 边界判断

### 判断一：三买确认决策树

遇到疑似第三类买点信号，按以下决策树逐步确认：

```
步骤一：该级别走势中枢是否已形成（前三段次级别走势存在价格区间重叠，ZD/ZG已确定）？
  ✅ 已形成 → 步骤二
  ❌ 未形成 → 不构成第三类买点，结构不足5段次级别

步骤二：是否存在完整的次级别向上离开段（离开段作为次级别走势类型已完成，整体高于ZG）？
  ✅ 存在，且离开段次级别走势类型完整（内部三段次次级别完成） → 步骤三
  ❌ 离开段次级别走势类型尚未完成 → 等待，不可提前介入（防骗线）

步骤三：当前回试是否为该次离开后的第一次次级别回试？
  ✅ 是第一次次级别回试 → 步骤四
  ❌ 之前已发生过一次完整次级别回试（第一次已回到中枢内或已结束） → 不构成三买

步骤四：回试次级别走势是否已完成，且低点 ≥ ZG（不跌破ZG）？
  ✅ 完成且低点 ≥ ZG → ✅ 第三类买点确认，买点 = 该回试低点
  ❌ 低点跌破ZG（低点 < ZG） → 三买失效，回到中枢震荡，等待新的判断
  ⏳ 回试次级别走势尚未完成 → 等待，不可提前冲入

步骤五（后验）：上述ZG是否来自最近那个中枢（而非历史旧中枢）？
  ✅ 是最近中枢的ZG → 有效
  ❌ 是更早历史中枢的ZG → 无效，须以最近中枢重新判断
```

### 判断二：三买 vs 二买的边界（重合条件）

三买与二买发生重合的**唯一充要条件**：

```
同时满足以下全部：
  A：第一类买点已出现
  B：一买后，次级别走势凌厉直接上破前下跌趋势的最后一个中枢（无中枢震荡过程）
  C：上破后产生次级别回抽，且回抽低点不触及该中枢（低点 ≥ ZG）

→ 此时该回抽低点同时是：
   - 第二类买点（一买后第一次次级别回调的低点）
   - 第三类买点（次级别离开该中枢后回试不破ZG）
```

非重合情况：
- 正常路径（一买→中枢震荡→二买，回调低点跌入ZG以下）：无三买条件，不重合
- 中枢震荡后独立出现三买（已有完整中枢形成过程）：单纯三买，不与二买重合

来源：课21（0428）

---

### 判断三：针对最近中枢（多中枢情况）

遇到走势中有多个历史中枢需确定三买基准：

```
步骤一：列出当前级别所有已完成的走势中枢，按时间排序
步骤二：取时间上最近（最新形成）的那个中枢，读取其 ZG（买点判断）或 ZD（卖点判断）
步骤三：
  回试低点 ≥ 最近中枢的ZG → ✅ 有效三买
  回试低点 < 最近中枢的ZG，但 ≥ 某历史旧中枢的ZG → ❌ 无效（跌破了最近中枢ZG）
```

典型错误：以某历史高点或旧中枢的ZG为基准，认定"回抽未突破某旧高"即为三买。正确做法是**只**参照最近中枢的ZG。——课58（0545）

### 判断四：中枢震荡 vs 三买的区分

遇到中枢延伸中反复震荡与三买的混淆：

```
离开后回试：低点是否重新进入中枢区间（低点 < ZG）？
  低点 < ZG（跌回中枢）→ 这是中枢震荡，不是三买；中枢延伸未结束
  低点 ≥ ZG（不破ZG）→ 三买成立

特殊情况（两次回试）：
  第一次回试进入中枢（低点 < ZG）→ 延伸继续
  第二次回试不进入中枢（低点 ≥ ZG）→ 不构成三买（已非"第一次"回试）
  原则："三买和中枢延伸的结束是一回事情，关于中枢的延伸结束，前面都有严格的数学定义，可参考。"——课38（0485）
  <!-- 整理者注：应用于判断四的原则是，第一次回试进入中枢（低点 < ZG）就意味着延伸仍未结束 -->
```

### 判断五：回抽走势复杂性处理

遇到回试段内部结构复杂（非简单单次回调）的判断：

```
回试次级别走势是三角形5段或其他复杂结构？
  → 只要整个回试次级别走势类型完成后，最终低点 ≥ ZG → 仍构成三买
  → 注意：C段不一定创新低，复杂回拉中看整体次级别走势是否完成，而非单笔低点
  来源：课24（0436）——"第三类买点的结束位置不一定是整个回拉的最低位置。"
```

### 判断六：小级别三买 vs 对应操作级别三买的选择

遇到大级别中枢突破但需要等到大级别三买时间成本过高：

```
大级别三买是否满足充要条件（次级别离开+次级别回试完成）？
  ✅ 已满足 → 按对应操作级别三买介入，最稳妥

  ❌ 等待中（如周线三买需等日线级别走势，时间成本极高）→ 可降级：
    判断该突破是否为"大幅快速波动+成交量配合的猛烈走势"：
      是 → 可用小级别（30分钟/5分钟）三买提前介入
      否 → 不可降级，与普通中枢震荡难以区分
    特别提示：卖点宜按小级别，决不卖晚；买点可稍等大级别，安全优先
  来源：课54（0535）
```

## 💡 操作规则

### 操作规则一：三买确认买入规则

前提：[某级别走势中枢已形成，ZG已确定] AND [完整次级别向上离开段已完成] AND [回试次级别走势正在进行或已完成]
触发：[回试次级别走势完成（内部次次级别三段走势结束），最低点 ≥ ZG，且为第一次次级别回试]
操作：[确认回试低点 ≥ ZG，次级别走势类型完整] → [在回试低点附近买入（不追涨，等回调形成后买）] → [止损设于 ZG 下方（跌破ZG则三买失效，无条件止损）] → [按三买买卖法后续步骤管理仓位]
冲突优先级：三买属于最晚介入的安全买点（一买最早、二买次之、三买最晚），与一买/二买不冲突（分属不同时间节点）；三买确认前不追涨介入（次级别走势未完成时不冲入）。

### 操作规则二：三买买卖法六步骤（散户核心操作程序）

前提：[已选定操作级别（建议30分钟或日线，不宜过小）] AND [持续监控该级别下的走势中枢与离开信号]
触发：[在选定级别出现第三类买点（充要条件全部满足）]
操作：

> 原文引用（课79，0682）：
> "1、选定一个足够去反应的级别，例如，30分钟或5分钟的，或者干脆就用日线级别的。
> 2、只介入在该级别出现第三类买点的股票。
> 3、买入后，一旦新的次级别向上不能新高或出现盘整背驰，坚决卖掉。
> 4、如果股票没出现3的情况，那一定是进入新一轮该级别的中枢上移中，一定要持有到该上移的走势出现背驰后至少卖掉一半，然后一个次级别下来，在一个次级别上去，只要不创新高或盘整背驰，就一定要把所有股票出掉。
> 5、尽量只介入第一个中枢的第三类买点。因为第二个中枢以后，形成大级别中枢的概率将急促加大。
> 6、本方法，一定不能对任何股票有感情，所有股票，只是烂纸，只是用这套有效方法去把纸变黄金。"
>
> ——课79（0682）

冲突优先级：步骤4为经验性指引（原文：「尽量」）——第二个中枢以后形成大级别中枢概率大增，应审慎参与而非绝对不参与；若后续出现更高级别中枢位置的三买信号，按定理二情况A（趋势继续）持有，按情况B（大级别中枢）执行步骤3。

### 操作规则三：三类买卖点综合配合操作

前提：[完整跟踪某级别从第一类买点开始的走势演化]
触发：[第三类买点出现]
操作：[在一买/二买处先建仓] → [持有并观察是否出现三买] → [三买出现则确认继续持有，向上趋势开启] → [三买未出现（中枢持续震荡）则按中枢震荡操作（卖点减仓、买点回补）]；简化版：[在突破的次级别走势出现背驰时先出，然后观察三买是否出现，出现则回补，不出现则不追]
冲突优先级：三买出现前不主动清仓（中枢震荡阶段维持仓位）；三买失效（跌破ZG）时无条件止损；来源：课29（0456）。

### 操作规则四：大级别走势背景下三买操作意义判断

前提：[小级别（如5分钟）三买出现，但需评估大级别背景]
触发：[出现小级别三买信号]
操作：
  [判断大级别走势状态]
  → 大级别中枢上移（上涨趋势）：小级别三买警戒意义，不改变大格局，可参与但注意大级别节奏

  → 大级别中枢震荡：判断小级别三买是否会延伸出大级别三买；无此延伸则仅为短线震荡机会
