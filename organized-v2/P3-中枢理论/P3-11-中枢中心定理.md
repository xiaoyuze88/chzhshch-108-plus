# P3-11 中枢中心定理

> **文件用途说明**：本文件是面向 AI 系统的缠论知识点文件，内容须自洽、无歧义、可机器解析。
> 所有定义优先使用充要条件格式，经验性判断须单独标注并与精确定义区分。

## 概述

中枢中心定理是缠论 P3-中枢理论的核心公理化组件，在课20（0424）集中给出数学表述。本章构建了描述中枢内部状态与跨中枢关系的完整数学语言：六价体系（ZG/ZD/GG/G/D/DD）、中心定理一（延伸等价条件）、中心定理二（双中枢四分类判断），并由定理一直接推导出第三类买卖点定理。课49（0515）在此基础上给出操作框架与两大利润最大定理；课73（0653）给出中枢上移与中枢震荡两种获利模式的宏观分类。课30（0459）补充了中枢扩展的精确数学界定。

## 📖 定义

### 定义一：Z走势段（Z-segment）

定义：当且仅当某次级别走势类型满足以下条件时，称为该走势中枢的 **Z走势段**（记为 Zn，n 为时间顺序编号）：
- [条件1] 该次级别走势类型位于一个已形成（ZD < ZG）的走势中枢的围绕运动序列中
- [条件2] 该次级别走势类型的运动方向与该走势中枢的形成方向一致

各 Zn 的高点记为 gn，低点记为 dn；不满足条件2的次级别走势类型称为反向连接段，不计入 Zn 序列。

> 原文引用："以后都把这些与中枢方向一致的次级别走势类型称为Z走势段，按中枢中的时间顺序，分别记为Zn等，而相应的高、低点分别记为gn、dn。"
>
> ——课20（0424）

---

### 定义二：中枢六价体系（ZG / ZD / GG / G / D / DD）

定义：在走势中枢已成立（ZD < ZG）且 Z走势段序列已定义的前提下，对 n 遍历中枢内所有 Zn，定义六个价格指标：

| 符号 | 定义公式 | 语义 |
|------|----------|------|
| **ZG** | min(g1, g2) | 中枢上沿（前两个 Z段高点的最小值，固定于中枢初始形成时） |
| **ZD** | max(d1, d2) | 中枢下沿（前两个 Z段低点的最大值，固定于中枢初始形成时） |
| **GG** | max(gn)，n遍历全部Zn | 中枢内所有 Z段高点的历史最大值（随延伸动态更新） |
| **G**  | min(gn)，n遍历全部Zn | 中枢内所有 Z段高点的历史最小值（随延伸动态更新） |
| **D**  | max(dn)，n遍历全部Zn | 中枢内所有 Z段低点的历史最大值（随延伸动态更新） |
| **DD** | min(dn)，n遍历全部Zn | 中枢内所有 Z段低点的历史最小值（随延伸动态更新） |

**四价不等式约束（正常延伸中枢的数学必要条件）**：

$$DD \leq ZD \leq ZG \leq GG$$

证明：
- DD = min(dn) ≤ d1 且 ≤ d2，故 DD ≤ max(d1, d2) = ZD ✓
- ZD < ZG：中枢存在的充要条件保证（严格不等号）✓
- ZG = min(g1, g2) ≤ g1 ≤ max(gn) = GG ✓

> 原文引用："定义四个指标，GG=max(gn)，G=min(gn)，D=max(dn)，DD=min(dn)，n遍历中枢中所有Zn。特别地，再定义ZG=min(g1、g2)，ZD=max(d1、d2)，显然，[ZD，ZG]就是缠中说禅走势中枢的区间。"
>
> ——课20（0424）

### 定义三：a+A+b+B+c 当下分析框架

定义：当分析某级别走势的当下状态时，采用如下标准化标注框架（小写字母=次级别走势段，大写字母=该级别走势中枢）：

| 符号 | 类型 | 含义 |
|------|------|------|
| **a** | 次级别走势段（小写） | 进入首个中枢 A 前的最后一段次级别运动 |
| **A** | 走势中枢（大写） | 第一个操作级别走势中枢，区间 [ZD_A, ZG_A]，拥有完整六价体系 |
| **b** | 次级别走势段（小写） | 离开 A 并可能进入 B 的连接段（方向决定趋势方向；若低点不破 ZG_A 则为第三类买点段） |
| **B** | 走势中枢（大写） | 第二个操作级别走势中枢，区间 [ZD_B, ZG_B]（若已形成） |
| **c** | 次级别走势段（小写） | 当下正在发展的最新次级别走势段 |

框架扩展规则：
- 超过两个中枢时，以此类推：a + A + b + B + c + C + d + ...
- 当前分析重心为"当下段（最新小写字母）"相对于"最后一个大写中枢"的位置关系。
- ZD_A/ZG_A/GG_A/DD_A 用于定理一检验（Zn 区间是否与 [ZD_A,ZG_A] 重叠）。
- ZG_A vs. ZD_A 与 GG_B vs. DD_A（即后ZG/后ZD与前DD/前GG的关系）用于定理二检验。

## 📐 定理

### 定理一：中枢中心定理一（延伸等价条件）

前提：[走势中枢已成立，六价体系已建立] AND [当前存在某 Z走势段 Zn，其区间为 [dn, gn]]

→ 结论（双向等价，互为充要）：

**中枢处于延伸中** ⟺ **任意 Zn 的区间 [dn, gn] 与 [ZD, ZG] 有重叠**
（即对所有 n，均满足 dn ≤ ZG 且 gn ≥ ZD）

逆命题：
**中枢延伸结束** ⟺ **存在某 Zn，使得 dn > ZG（向上突破中枢上沿）或 gn < ZD（向下跌破中枢下沿）**

延伸结束后的必然结果（二选一，互斥）：
- 产生更高级别的走势中枢（触发定理二的条件3或4）
- 趋势及其延续（触发定理二的条件1或2）

> 原文引用："走势中枢的延伸等价于任意区间[dn，gn]与[ZD，ZG]有重叠。换言之，若有Zn，使得dn>ZG或gn<ZD，则必然产生高级别的走势中枢或趋势及延续。"
>
> ——课20（0424）

---

### 定理二：中枢中心定理二（双中枢四分类判断）

前提：[前后同级别存在两个走势中枢，分别记为"前中枢"和"后中枢"；各自的 GG/DD/ZG/ZD 均已确定]

→ 结论（四路穷举，互斥完备）：

| 条件 | 等价含义 | 中枢位置关系 |
|------|---------|-------------|
| **后GG < 前DD** | 下跌及其延续 | 后中枢完全在前中枢下方（无任何波动重叠） |
| **后DD > 前GG** | 上涨及其延续 | 后中枢完全在前中枢上方（无任何波动重叠） |
| **后ZG < 前ZD 且 后GG ≥ 前DD** | 形成高级别走势中枢 | 两中枢 ZG/ZD 区间不重叠，但极端波动有重叠 |
| **后ZD > 前ZG 且 后DD ≤ 前GG** | 形成高级别走势中枢 | 两中枢 ZG/ZD 区间不重叠，但极端波动有重叠 |

注意：条件3和条件4均属于"围绕两中枢产生的波动区间有重叠"，触发走势级别延续定理二（见定理五）。

> 原文引用："前后同级别的两个缠中说禅走势中枢，后GG < 前DD等价于下跌及其延续；后DD > 前GG等价于上涨及其延续。后ZG < 前ZD且后GG >= 前DD，或后ZD > 前ZG且后DD =< 前GG，则等价于形成高级别的走势中枢。"
>
> ——课20（0424）

---

### 定理三：第三类买卖点定理（由定理一直接推导）

前提：[走势中枢 A 已形成，ZG_A 和 ZD_A 已确定] AND [已有一个次级别走势类型向上（或向下）完整离开该中枢]

→ 结论：

**第三类买点**：当且仅当满足：
- [条件1] 向上离开中枢为**第一次**离开
- [条件2] 随后以一个完整次级别走势类型回试
- [条件3] 回试的低点不跌破 ZG_A（即 低点 ≥ ZG_A）
→ 该回试低点构成第三类买点。

**第三类卖点**：当且仅当满足：
- [条件1] 向下离开中枢为**第一次**离开
- [条件2] 随后以一个完整次级别走势类型回抽
- [条件3] 回抽的高点不升破 ZD_A（即 高点 ≤ ZD_A）
→ 该回抽高点构成第三类卖点。

**条件3的定理一依据**：回试低点 ≥ ZG_A，意味着该回试 Zn 的 dn 未 > ZG_A，定理一延伸条件依然可成立（中枢未被反向突破），当下位于中枢上方，为安全买入区域。

> 原文引用："由定理一，可以得到第三类买卖点定理：一个次级别走势类型向上离开缠中说禅走势中枢，然后以一个次级别走势类型回试，其低点不跌破ZG，则构成第三类买点；一个次级别走势类型向下离开缠中说禅走势中枢，然后以一个次级别走势类型回抽，其高点不升破ZD，则构成第三类卖点。"
>
> ——课20（0424）

---

### 定理四：走势级别延续定理一

前提：[当前级别走势中，更大级别走势中枢尚未产生]
→ 结论：该级别走势类型将延续（只能是含该级别中枢的盘整延续或趋势延续，不产生级别跳变）。

推论：要使某级别走势改变，**必须先使更大级别走势中枢产生**（即触发定理五）。

> 原文引用："在更大级别缠中说禅走势中枢产生前，该级别走势类型将延续。"
>
> ——课20（0424）

---

### 定理五：走势级别延续定理二（充要条件）

前提：（无需预设，为充要条件命题）
→ 结论（双向等价）：

**更大级别走势中枢产生** ⟺ **围绕连续两个同级别走势中枢产生的波动区间产生重叠**
（即定理二中条件3或条件4成立）

> 原文引用："更大级别缠中说禅走势中枢产生，当且仅当围绕连续两个同级别缠中说禅走势中枢产生的波动区间产生重叠。"
>
> ——课20（0424）

### 定理六：缠中说禅第一利润最大定理

前提：[固定操作品种] AND [确定操作级别 L] AND [操作模式为以下组合：
  (a) 在操作级别中枢震荡中做差价（中枢上方减仓，中枢下方增仓，成本降至0后增加筹码）
  (b) 在中枢第三类买点后持股不做差价，直到形成新的 L 级别中枢
  (c) 在新中枢继续（a）的震荡差价
  (d) 在中枢完成向上移动出现背驰后，抛出全部筹码]
→ 结论：以上操作模式在"平均利润率"（总利润/总时间）维度上最大。

> 原文引用："对于任何固定交易品种，在确定的操作级别下，以上缠中说禅操作模式的利润率最大。"
>
> ——课49（0515）

---

### 定理七：缠中说禅第二利润最大定理

前提：[不同交易品种] AND [确定操作级别 L] AND [激进操作模式：
  只在第三类买点买入，一旦形成新的 L 级别中枢就退出换股，
  不参与中枢震荡差价，不等待背驰出现]
→ 结论：以上激进操作模式在"每次操作平均利润/占用资金时间"维度上最大。

注意：第一定理的利润率定义为"总体收益"，第二定理的利润率定义为"单次效率"；第二定理需要更强的市场条件（必须持续有可操作的第三类买点出现）；两者不可偏废。

> 原文引用："对于不同交易品种交易中，在确定的操作级别下，以上激进的缠中说禅操作模式的利润率最大。"
>
> ——课49（0515）

## 🔍 边界判断

### 当下位置三路判断（a+A+b+B+c 框架定位）

遇到"判断当下价格相对于最后一个操作级别中枢的位置" →
  先判断"是否已确认一个操作级别中枢 A"：
    否 → 等待中枢形成（三段次级别走势有重叠区间，即 ZD < ZG）
    是 → 判断"当下价格区间与 [ZD_A, ZG_A] 的关系"：
      当下价格在 [ZD_A, ZG_A] 内 → **情形1：中枢延伸中**（不操作或做震荡差价）
      当下价格在 ZG_A 上方 → 判断"是否已出现过该中枢的第三类买点"：
        未出现（b段离开后首次回试尚未完成）→ **情形3a：等待回试确认第三类买点**
        已出现（第三类买点已确认，当下为 c 段）→ **情形3b：中枢上移/趋势进行中，满仓持有**
      当下价格在 ZD_A 下方 → 判断"是否已出现过该中枢的第三类卖点"：
        未出现 → **情形2a：等待回抽确认第三类卖点**
        已出现 → **情形2b：中枢下移/下跌趋势，寻找背驰买点**

---

### 中枢延伸 vs. 级别扩张 vs. 趋势 的区分判断

遇到"某中枢已震荡超过三段次级别走势，判断后续方向" →
  检验定理一：判断"是否存在某 Zn 使得 dn > ZG 或 gn < ZD"：
    否（所有 Zn 区间均与 [ZD,ZG] 有重叠）→ **中枢依然在延伸中**，继续震荡差价
    是（定理一延伸条件被打破）→ 检验定理二：判断"两中枢（A与新生中枢B）的波动区间是否产生重叠"：
      是（满足条件3或4）→ **形成更高级别走势中枢**（级别扩张），按新级别重新分析
      否（满足条件1）→ **下跌趋势**（后GG < 前DD），执行卖出或等待背驰买点
      否（满足条件2）→ **上涨趋势**（后DD > 前GG），第三类买点后满仓持有

关键约束（来自课30）：**中枢扩展有精确数学定义，在一个中枢还延伸时，只有一个中枢，不存在"中枢1"与"中枢2"并存的情况**——只要 Zn 区间与 [ZD,ZG] 仍有重叠，该中枢就还没结束，不能拆分为两个中枢。

---

### 第三类买点 vs. 中枢震荡回调 的区分

遇到"价格从中枢上方回落，判断是否为第三类买点" →
  先判断"是否为该中枢第一次离开后的第一次回试"：
    否（此前已出现过第三类买卖点）→ **不构成第三类买点**（第三类买卖点必须是第一次离开后的第一次回试）
    是 → 判断"回试次级别走势是否已完成"：
      否（回试走势尚在进行中）→ 等待次级别走势完成确认
      是 → 判断"回试低点是否 ≥ ZG"：
        是（低点 ≥ ZG）→ **构成第三类买点**（定理三成立，可按次级别精确定位介入）
        否（低点 < ZG）→ **不构成第三类买点**，中枢延伸条件被打破，重新判断结构

---

### 中枢震荡获利模式 vs. 中枢上移获利模式 的切换判断

遇到"判断当前仓位策略：震荡差价还是满仓持有" →
  判断"是否已出现该操作级别中枢的第三类买点"：
    否（中枢依然在震荡延伸中）→ **采用中枢震荡模式**：
      中枢上方（ZG以上）→ 减仓卖出
      中枢下方（ZD以下）→ 加仓买入
      前提：中枢震荡依旧（所有 Zn 区间与 [ZD,ZG] 有重叠）
    是（第三类买点已出现）→ **切换为中枢上移模式**：
      满仓持有，不做中枢震荡差价
      中枢上移期间出现背驰 → 全部清仓，等待下一买点
    注意：中枢震荡中出现的"类盘整背驰力度衰减"≠ 中枢完成向上移动中的背驰，两者在**第三类买点前后**分别判断，不可混淆。

## 💡 操作规则

### 操作规则一：中枢震荡差价操作

前提：[已确定操作级别 L] AND [识别出最后一个 L 级别中枢，中枢延伸中（定理一条件成立，未出现第三类买卖点）]
触发：[价格到达中枢上方（Zn 高点区域）或中枢下方（Zn 低点区域）]
操作：[在 ZD 以下区域增加仓位（买入）] → [在 ZG 以上区域减少仓位（卖出）] → [用 MACD 辅助判断当前震荡力度与前一同向震荡力度比较，预判是否可能出现第三类卖点] → [若力度判断提示背驰风险，提前控制仓位]
冲突优先级：当出现第三类卖点信号时，本规则**让位于**"清仓规则"；本规则仅在确认中枢延伸有效时执行，一旦 Zn 突破 ZG（或跌破 ZD）立即按定理一重新判断。

---

### 操作规则二：第三类买点介入与持仓规则

前提：[当前价格在操作级别中枢 A 的 ZG_A 上方] AND [为该中枢第一次次级别离开后的第一次回试] AND [回试次级别走势已完成]
触发：[确认回试低点 ≥ ZG_A，第三类买点成立]
操作：[在第三类买点次级别走势完成时精确买入（按次级别乃至再次级别的区间套定位）] → [转入满仓持有模式，不再做震荡差价] → [等待新的 L 级别中枢 B 形成，继续规则一] → [若中枢 B 完成向上移动出现 L 级别背驰，全部卖出]
冲突优先级：第三类买点后若出现更大级别盘整（未形成趋势），在盘整顶部（次级别背驰点）需减仓或清仓；本规则覆盖规则一（中枢上移期间无差价操作）。

### 操作规则三：当下位置定位五步流程（a+A+b+B+c 框架应用）

前提：[新进入某品种或需要重新判断介入时机]
触发：[打开走势图需要判断当下位置与操作策略]
操作：
  [步骤1：确认自身操作级别 L] →
  [步骤2：找出该 L 级别最后一个已完成的走势中枢 A，确认 ZD_A / ZG_A / GG_A / DD_A] →
  [步骤3：判断当下价格相对 A 的位置（情形1/2/3，详见边界判断）] →
  [步骤4（情形3b）：若已出现第三类买点且距离不远，评估次级别回调是否已形成买点，可介入；若第三类买点后已出现次级别顶背驰，等待] →
  [步骤5（情形2b）：若在 A 下方且第三类卖点已出现，分析当前向下走势的次级别结构，寻找背驰买点，不追跌]
冲突优先级：**不参与操作级别以上级别的下跌与超过操作级别的盘整**（核心纪律）；技术精度不足时，只把握自身能把握的机会，宁可错过不可操错。

## 📜 经验判断

> ⚠️ **原文未给出充要条件**，以下为原文经验性指引，AI 使用时须视为参考倾向而非确定规则。

| 场景 | 判断倾向（原文指引） | 边界（不适用情形） |
|------|-------------------|--------------------|
| 场景：中枢后向上一段（离开中枢后无背驰），估计后续幅度 | 判断倾向：若不出现背驰段，将形成第二个中枢；若第二个中枢级别比第一个低，上涨动能较强，不必过早出场 | 边界：已出现明确背驰信号时不适用，应按背驰清仓规则处理；本条仅用于出场时机的经验估算 |
| 场景：中枢震荡中某段出现"类盘整背驰"力度衰减，判断是否卖出 | 判断倾向：第三类买点出现**之前**的力度衰减属于中枢内正常收敛，不代表中枢结束；买点出现**之后**的调整属于中枢上移过程，不做空 | 边界：若力度衰减伴随价格跌破 ZG，则不属于正常中枢内收敛，应按定理一重新判断 |
| 场景：走势中第一、二、三类买点完美依次出现 | 判断倾向："基本比较厉害"——后续上涨动能通常较强 | 边界：为经验性倾向，非数学保证；不适用于买点不完整（缺少某一类）的走势 |
| 场景：在中枢震荡中判断是否能做差价 | 判断倾向：严格意义上，只有**围绕操作级别中枢的震荡差价**是最安全的（理论上必然能做出来且不丢失筹码）；中枢完成后向上移动时的差价不应做 | 边界：对于技术精度不足者，中枢震荡也可出现第三类卖点导致无法回补，应预留安全余量 |
