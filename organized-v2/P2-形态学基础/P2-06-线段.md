# P2-06-线段（特征序列与划分标准）

> **文件用途说明**：本文件是面向 AI 系统的缠论知识点文件，内容须自洽、无歧义、可机器解析。
> 所有定义优先使用充要条件格式，经验性判断须单独标注并与精确定义区分。

## 概述

线段是缠论形态学的核心结构单元，由至少三笔组成，是构建中枢与走势类型的最小分析基础部件。本章定义了线段的构成条件、特征序列及其两种破坏情况，为中枢计算和走势类型分解提供必要前提。原文在课45至课84间经历多次修订（主要修订节点：课65、课67、课71、课77、课78、课81），最终版本以课84综合所有修订为基准。

**前置依赖**（自包含定义）：
- **笔**：一个顶分型和一个底分型之间（顶底间至少一根独立K线），先底后顶=向上笔，先顶后底=向下笔
- **顶分型**：相邻三根K线（经包含处理后），中间K线最高点高于左右两侧K线最高点
- **底分型**：相邻三根K线（经包含处理后），中间K线最低点低于左右两侧K线最低点
- **包含关系处理**：K1高点>=K2高点 AND K1低点<=K2低点（或反向），上升中取较高高点和较高低点，下降中取较低高点和较低低点合并为一根

## 📖 定义

### 定义一：线段（综合课65~课84，最终有效版本）

**定义**：当且仅当满足以下**全部**条件时，构成一条线段：

1. **三笔前提**：由至少三笔组成（上下上：向上笔-向下笔-向上笔；或下上下：向下笔-向上笔-向下笔）；线段中包含的笔数**必须为奇数**（3、5、7…）。
2. **重叠前提**：前三笔之间必须存在价格区间重叠部分（第一笔与第三笔的价格区间有交集）。
3. **终结条件**：线段的终结通过特征序列出现对应分型确认——满足第一种情况（无缺口式）或第二种情况（有缺口式）之一即可（见定义三、定义四）。
4. **破坏原则**：线段必须被另一条**反向**线段破坏，单笔不能构成有效破坏；被同方向线段"破坏"不成立。

不满足任一条件（笔数不足三、无重叠、特征序列无分型、仅被单笔或同向线段突破）则不构成线段终结。

> "线段被破坏，当且仅当至少被有重叠部分的连续三笔的其中一笔破坏。而只要构成有重叠部分的前三笔，那么必然会形成一线段，换言之，线段破坏的充要条件，就是被另一个线段破坏。"
>
> ——课65（0593）

> "线段划分的最基本原则，就是线段必须至少有三笔……线段中包含笔的数目，都是单数的。而且，线段开始的那三笔，必须有重合，开始三笔没有重合的，是构不成线段的。另外，线段必须被线段所破坏才能确定其完成。"
>
> ——课77（0675）

### 定义二：特征序列（课67，最终定义）

**定义**：在线段分析中，取与线段方向相反的笔构成的序列，称为该线段的**特征序列**：

- 以**向上笔开始**的线段：特征序列 = 其中所有**向下笔**，记为 `X1 X2 … Xn`
- 以**向下笔开始**的线段：特征序列 = 其中所有**向上笔**，记为 `S1 S2 … Sn`

特征序列元素可进行**包含关系处理**（规则同K线包含处理）；处理后称为**标准特征序列**。以后无特别说明，特征序列均指标准特征序列。

**缺口**：标准特征序列中，相邻两元素之间若**无价格区间重叠**，称为该序列的一个缺口。

**分型方向约束**：
- 以向上笔开始的线段：**只考察特征序列的顶分型**
- 以向下笔开始的线段：**只考察特征序列的底分型**

> "序列`X1X2…Xn`称为以向上笔开始线段的特征序列；序列`S1S2…Sn`称为以向下笔开始线段的特征序列。特征序列两相邻元素间没有重合区间，称为该序列的一个缺口。"
>
> ——课67（0614）

---

### 定义三：线段破坏——第一种情况（无缺口式）

**定义**：当且仅当同时满足以下条件时，线段以**第一种情况**终结：

1. 标准特征序列中出现分型（上升线段出现**顶分型** / 下降线段出现**底分型**）
2. 构成该分型的**第一元素与第二元素之间不存在缺口**

→ 结论：该线段在该分型的极值点（顶分型高点 / 底分型低点）处终结，该极值点为线段终点。

> "特征序列的顶分型中，第一和第二元素间不存在特征序列的缺口，那么该线段在该顶分型的高点处结束，该高点是该线段的终点。"
>
> ——课67（0614）

---

### 定义四：线段破坏——第二种情况（有缺口式）

**定义**：当且仅当同时满足以下条件时，线段以**第二种情况**终结：

1. 标准特征序列中出现分型（上升线段出现**顶分型** / 下降线段出现**底分型**）
2. 构成该分型的**第一元素与第二元素之间存在缺口**
3. 从该分型极值点开始的**反向笔出发**，构建**第二特征序列**，其中出现相应分型（**不再区分第一/第二种情况**，只要出现分型即可）

→ 结论：原线段在原分型极值点处终结。

**关键约束**：
- 第二特征序列的分型判断**必须严格按包含关系处理**
- 第二特征序列不必封闭第一特征序列的缺口
- 第二特征序列**不再区分**一/二种情况，只要出现分型即终结确认

> "特征序列的顶分型中，第一和第二元素间存在特征序列的缺口，如果从该分型最高点开始的向下一笔开始的序列的特征序列出现底分型，那么该线段在该顶分型的高点处结束。强调，在第二种情况下，后一特征序列不一定封闭前一特征序列相应的缺口，而且，第二个序列中的分型，不分第一二种情况，只要有分型就可以。"
>
> ——课67（0614）

## 📐 定理

### 定理一：缠中说禅线段分解定理

前提：[一条线段 A 已形成，后续出现破坏走势]
→ 结论：[线段 A 终结的充要条件是被另一条有重叠的反向线段破坏；单笔破坏线段端点不构成线段终结]

> "线段被破坏，当且仅当至少被有重叠部分的连续三笔的其中一笔破坏。线段破坏的充要条件，就是被另一个线段破坏。"
>
> ——课65（0593）

---

### 定理二：线段笔数奇数定理

前提：[任意一条有效线段]
→ 结论：[该线段包含的笔数必然为奇数（3、5、7…）]

推导：线段从底（或顶）开始，终结于顶（或底），方向相反；笔交替顶底，从底到顶（或顶到底）跨越奇数笔。

> "线段中包含笔的数目，都是单数的。"
>
> ——课77（0675）

---

### 定理三：线段划分唯一性定理

前提：[给定同一级别K线图，精度标准统一]
→ 结论：[所有线段可以唯一地连接整张走势图，不存在两种不同的合法划分]

> "按照这个划分，一切同一级别图上的走势都可以唯一地划分为线段的连接，正如一切同一级别图上的走势都可以唯一地划分笔的连接。"
>
> ——课67（0614）

---

### 定理四：线段破坏方向性定理

前提：[线段 A 以向上笔开始（向上线段）]
→ 结论：[破坏线段 A 的线段必须以向下笔开始；以向上笔开始的线段不能破坏另一以向上笔开始的线段]

> "线段被线段破坏，必须不能是被同一性质的线段所破坏，也就是从向上一笔开始的线段不可能被向上一笔的线段所有破坏，必然是被从向下一笔开始的线段所破坏。"
>
> ——课77（0675）

---

### 定理五：线段破坏传递性定理

前提：[线段序列中，线段 C 破坏线段 B] AND [线段 B 破坏线段 A]
→ 结论：[线段的破坏关系可以逆时间传递；若划分不能保证每一段都被后一段破坏，则划分有误]

> "除非是新股上市后最开始的一段，否则任何一段都是破坏前一段的……线段的破坏是可以逆时间传递的，也就是说被后线段破坏的线段，一定破坏前线段，如果违反这个原则，那线段的划分一定有问题。"
>
> ——课78（0677）

---

### 定理六：线段端点方向定理

前提：[任意一条有效线段]
→ 结论：
  - 以向上笔开始的线段必须以向上笔结束（终点为顶分型）
  - 线段不可能从底到底，也不可能从顶到顶
  - 线段两端，顶分型极值**必须高于**底分型极值；否则划分有误

> "以向上笔开始的线段一定结束于向上笔，不可能一个线段，开始是向上笔，结束于一个向下笔……一个线段，不可能是从底到底或从顶到顶。同一线段中，两端的一顶一底，顶肯定要高于底，如果你划出一个不符合这基本要求的线段，那肯定是划错了。"
>
> ——课78（0677）

---

### 定理七：第二种情况数学完备性定理

前提：[连续出现一串"第二种情况"的线段破坏（线段 B 对 A 是第二种情况，C 对 B 也是…）]
→ 结论：[该串必然在有限步内出现第一种情况的破坏；数学上可由区间收敛性保证（区间不可能无限缩小至点）；因此操作中第二特征序列不再分情况，只要有分型即可]

> "在一串的相对前一线段是第二种情况的线段串中，比如最终会出现第一种情况的破坏，这样倒推回来，必然有这一串假定线段间的连续破坏。正因为这样，所以在第二种情况中的第二特征序列判断中，就不再分第一、二种情况了，这样是免得有一串线段串不断收敛后倒推回来的麻烦。"
>
> ——课77（0675）

### 两种破坏情况判断树

遇到特征序列出现疑似分型时，判断线段是否终结：

```
Step 1: 确认标准特征序列中是否出现分型
    上升线段 → 看顶分型
    下降线段 → 看底分型
        无对应分型出现 → 线段未终结，继续延伸，END
        有对应分型出现 → 进入 Step 2

Step 2: 检查构成分型的第一元素与第二元素之间是否有缺口
    无缺口 → 【第一种情况（无缺口式）】：
                线段在该分型极值点终结 ✓  END
    有缺口 → 进入 Step 3

Step 3: 从该分型极值点开始，构建第二特征序列
         （注意：必须严格进行包含关系处理；不必封闭原缺口）
    第二特征序列出现对应分型（任何情况，不再区分一/二种情况）
        → 【第二种情况（有缺口式）】：原线段在原分型极值点终结 ✓  END
    第二特征序列无分型出现
        → 原线段未终结，继续延伸，END
```

### 特征序列包含关系适用边界

遇到两个特征序列元素是否可做包含关系处理的问题：

```
判断：两元素是否属于同一线段的同一特征序列？
    是 → 可进行包含关系处理（规则同K线处理）
    否（跨越不同线段/不同特征序列）→ 不可处理（无意义）

特殊情况——假设分界点两侧（第一种情况适用）：
    假设转折点前最后一个特征元素（第一元素）与转折点后第一笔（第二元素）：
        这两者属于"待定状态"，不能做包含关系处理
        原因：第一种情况下，跨越分界点两侧的包含处理会弱化转折力度的呈现

对比——第二种情况的第二特征序列内部：
    必须严格按包含关系处理
    原因：第二种情况本身意味着对原线段转折的能量不足，
          包含关系的出现意味着原线段能量充足，必须如实反映
```

### 古怪线段判断与处理

遇到"古怪"的线段（线段起点非最高/低点等情况）：

```
判断：此线段是否因"第一种情况笔破坏后未发展为线段破坏"所致？
    是 → 古怪线段客观存在，是合法的；此为线段古怪的唯一原因

具体路径（以向下线段 B 为例，出现向上笔破坏后）：
    向上笔完成后，跟随的向下走势：
        破了该向上笔的底（即该向上笔底点被跌破）
            → 原线段 B 未结束，仍在延续
            → 后续：若向上笔突破线段 B 高点 → 出现"起点非最高点"的古怪线段
        未破该向上笔的底
            → 向上笔可延伸出新线段，线段 B 被破坏 ✓
```

### 第二种情况三线段合并问题

遇到线段 A→B→C 序列中 C 未形成第二特征序列分型而直接新高/新低：

```
条件：
    线段 B 对线段 A 构成第二种情况（有缺口）破坏
    线段 C 没有形成第二特征序列的分型，直接新高或新低

判断：
    不能认为 A、B、C 是三个独立线段
    正确处理：A + B + C 合并为一个线段
```

---

### 缺口作为线段的处理

遇到缺口是否构成独立线段的问题：

```
判断缺口类型：
    缺口顺着走势方向，包含在某一笔内（未破坏该笔走势）
        → 该缺口是笔的组成部分，不单独算线段

    突发性逆向缺口（强势跳空，性质与一般走势不同）：
        形式上无三笔 → 可视为三个缺口的叠加（0=0+0+0）
        → 视为独立一段（非仅一笔）✓
```

---

### 精度统一原则

遇到极近高低点导致线段划分歧义：

```
原则：同一图表的所有线段划分必须统一精度标准（同一软件、同一小数位数）
    两点极为接近（差值在精度误差范围内）→ 可视为相等处理
    选择何种精度 → 对实质分析无影响，关键是全局一致性

若已明确精度标准后仍有歧义 → 两种划分均合法，对后续分析无实质影响
```

## 💡 操作规则

### 操作规则一：线段当下划分程序

前提：[已完成笔的划分] AND [当前K线已走出（当下性）]
触发：[出现疑似线段转折，需判断某转折点是否为线段分界点]
操作：
1. 假设该转折点为两线段的分界点
2. 提取：假设转折点前线段的**最后一个特征元素**（分型第一元素）
3. 提取：从该转折点开始的**第一笔**（分型第二元素）
4. 判断第一元素与第二元素间是否有缺口：
   - 无缺口 → 按第一种情况验证（直接确认终结）
   - 有缺口 → 按第二种情况验证（构建第二特征序列，看是否有分型）
5. 满足其中一种情况 → 确认为线段分界点 ✓
6. 不满足任何情况 → 原线段继续延伸，假设无效

冲突优先级：结果产生"古怪线段"时，依然有效；不因主观印象调整客观划分结果

---

### 操作规则二：线段标准化处理

前提：[线段划分完成] AND [需要以线段为基础构建中枢或走势类型分析]
触发：[发现线段内部存在最高点（或最低点）不在端点的情况]
操作：
1. 识别线段的实际最高点和最低点（可能不在起止端点）
2. 向上线段标准化：以实际最低点为起点，实际最高点为终点
3. 向下线段标准化：以实际最高点为起点，实际最低点为终点
4. 标准化后，所有线段的连接形成首尾相连的折线，作为构建中枢的基础部件

冲突优先级：标准化不影响线段破坏关系的判断；仅用于以线段为基础的后续分析层面

---

### 操作规则三：最低分析级别线段确认

前提：[开始对某级别K线图进行技术分析] AND [需确定最低分析级别]
触发：[选定K线周期，准备开始划分]
操作：
1. 确定最低分析级别（例如1分钟级别）
2. 该级别以下的所有波动，一律视为无内部结构的线段（最基础部件）
3. 在该最低级别视角下，相邻三段线段的重叠区间构成该级别的中枢
4. 最低级别的选择受操作成本、交易通道、资金规模等实际条件制约

冲突优先级：最低级别确定后，全程必须统一使用同一标准，不可中途更换

## 📜 经验判断

> ⚠️ **原文未给出充要条件**，以下为原文经验性指引，AI 使用时须视为参考倾向而非确定规则。

| 场景 | 判断倾向（原文指引） | 边界（不适用情形） |
|------|-------------------|--------------------|
| 场景：线段第三笔极其微弱，缩图后几乎看不到 | 判断倾向：该上下上/下上下属于更小级别的内部结构，不构成当前级别的线段 | 边界：若同一图中已有类似微弱程度的三折被认定为线段，则必须统一处理，不能选择性承认；选哪种标准关键是全程统一 |
| 场景：需判断线段结束的时机 | 判断倾向：线段结束与大级别走势段类似——趋势中用背驰确认，盘整中用盘整背驰确认，突发性事件看第二类买卖点 | 边界：此为操作层面经验性判断，不能替代特征序列分型的严格客观标准；特征序列分型才是充要条件 |
| 场景：线段幅度极短或极长 | 判断倾向：是否构成线段只看形态结构，与幅度/时间无关；任何低点均高于前一高点的走势无论延续多久均为一条线段 | 边界：仍需满足前三笔重叠的基本前提 |
| 场景：从图K线近期最高或最低点开始划分 | 判断倾向：实操中无需从上市第一天开始，选近期明确的最高/最低点出发，但对大走势认识会有局限 | 边界：如需完整分析某波行情，应从该波行情的明确起点（如大底）开始划分 |
