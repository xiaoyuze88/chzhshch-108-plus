# P5-17-盘整背驰 vs 趋势背驰

> **文件用途说明**：本文件是面向 AI 系统的缠论知识点文件，内容须自洽、无歧义、可机器解析。
> 所有定义优先使用充要条件格式，经验性判断须单独标注并与精确定义区分。

## 概述

盘整背驰 vs 趋势背驰是缠论 P5-背驰理论中最核心的区分关卡，也是从"幼儿园"到"小学毕业"的门槛。
本章精确定义盘整背驰与趋势背驰的构成条件、力度比较方法、背驰后走势约束（转折定理），并给出判断树。
原文在课24、课27初步建立框架，课29给出转折定理，课37给出趋势背驰内部结构精确约束，课43进行补习整合。

## 📖 定义

### 定义一：趋势背驰

定义：当 [某级别走势构成趋势（含 ≥2 个同向不重叠中枢）] 且 [在 a+A+b+B+c 结构中，A、B 为同级别中枢] 且 [c 为次级别走势类型（至少包含对 B 的一个第三类买卖点）] 且 [c 方向与 a 同向且创出新高（上涨）或新低（下跌）] 且 [c 段趋势力度 < a 段趋势力度（均线面积法或 MACD 面积对比）] 时，称为该级别的趋势背驰；不满足任一条件则不构成趋势背驰。

> 原文引用："没有趋势，没有背驰，不是任何 `a+A+b+B+c` 形式的都有背驰的。当说 `a+A+b+B+c` 中有背驰时，首先要 `a+A+b+B+c` 是一个趋势。而一个趋势，就意味着A、B是同级别的中枢，否则，就只能看成是其中较大中枢的一个震荡。"
>
> ——课37（0481）

> 原文引用（力度比较方法）："缠中说禅趋势力度：前一'吻'的结束与后一'吻'开始由短线均线与长期均线相交所形成的面积。在前后两个同向趋势中，当缠中说禅趋势力度比上一次缠中说禅趋势力度要弱，就形成'背驰'。"
>
> ——课15（0402）

---

### 定义二：盘整背驰

定义：当 [某级别走势处于盘整（单一中枢）中，或 a+A+b+B+c 结构中 A 与 B 级别不相等（A 级别大于 B，整体只能视为围绕 A 的震荡）] 且 [围绕中枢存在三段结构：第一段（A）+ 回调段（B）+ 第三段（C），A 与 C 方向相同] 且 [C 段的力度（MACD 面积或均线面积）< A 段的力度] 时，称为该级别的盘整背驰；不满足任一条件则不构成盘整背驰。

<!-- ⚠️ 附注：盘整背驰中对 A、C 两段自身的走势类型无额外约束——两段均可以是盘整（不要求为趋势）。 -->

> 原文引用："如果在第一个中枢就出现背驰，那不会是真正意义上的背驰，只能算是盘整背驰，其真正的技术含义，其实就是一个企图脱离中枢的运动，由于力度有限，被阻止而出现回到中枢里。"
>
> ——课27（0449）

> 原文引用："一般来说，第二、三类的买点，都有一个三段的走势，第三段往往都破点第一段的极限位置，从而形成盘整背驰，注意，这里是把第一、三段看成两个走势类型之间的比较，这和趋势背驰里的情况有点不同，这两个走势类型是否一定是趋势，都问题不大，**两个盘整在盘整背驰中也是可以比较力度的**。"
>
> ——课27（0449）

---

### 定义三：背驰段

定义：当 [某级别的某走势类型（盘整或趋势）已构成背驰或盘整背驰] 时，称该段走势类型为某级别的背驰段。

> 原文引用："在某级别的某类型走势，如果构成背驰或盘整背驰，就把这段走势类型称为某级别的背驰段。"
>
> ——课27（0449）

---

### 定义四：力度比较方法（多维对比）

以下三种判断方式，满足其一即可作为背驰信号，两种或以上同时满足为更标准信号：

**方法一（均线面积法）**：短线均线与长期均线相交形成的面积（"吻"间面积），C 段面积 < A 段面积。

**方法二（MACD 柱子面积法）**：MACD 红（或绿）柱子面积之和，C 段所有柱子面积总和 < A 段所有柱子面积总和。注意：须累加该段内所有柱子面积，不可仅比较单个柱子。

**方法三（黄白线未创新高/低法）**：若 C 段连新高（上涨情形）或新低（下跌情形）都创不出，则力度最弱，无需 MACD 辅助，直接判断为背驰。

> 原文引用："一般来说，只要其中一个符合就可以是一个背弛的信号，两个都满足就更标准了。"
>
> ——课27（0449）

> 原文引用（C段不创新高无需MACD）："C段不一定创新高...如果连新高都创不出，那力度就最弱，当然更不行。这时候，根本连MACD的辅助都没必要。MACD只要是辅助创新高的情况。"
>
> ——课38（0485）

## 📐 定理

### 定理一：缠中说禅背驰-转折定理

前提：[某级别趋势（≥2 个同向不重叠中枢）中，出现该级别的趋势背驰]
→ 结论：该趋势必然终止，且必然导致以下三种情况之一：
  1. 该趋势最后一个中枢的级别扩展（最弱情形，反弹仅触及最后中枢的 `DD=min(dn)`）
  2. 该级别更大级别的盘整（盘整中枢级别 > 原趋势中枢级别）
  3. 该级别以上级别的反趋势

> 原文引用："缠中说禅背驰-转折定理：某级别趋势的背驰将导致该趋势最后一个中枢的级别扩展、该级别更大级别的盘整或该级别以上级别的反趋势。"
>
> ——课29（0456）

> 原文引用（操作推论）："某级别的背驰必然导致该级别原走势类型的终止，进而开始该级别或以上级别的另外一个走势类型。"
>
> ——课43（0499）

---

### 定理二：趋势背驰回跌下限定理

前提：[趋势背驰成立（定理一前提满足）] AND [背驰后发生转折（非最弱的中枢级别扩展情形）]
→ 结论：回跌（或回升）一定至少重新回到 B 段（最后一个中枢）之内；此为操作可预知的最小跌幅（或最小涨幅）。

> 原文引用："背驰是最重要的，一旦出现背驰，其回跌，一定至少重新回到B段的中枢里。"
>
> ——课24（0436）

---

### 定理三：盘整背驰转折范围约束定理

**情形1**：前提：[盘整背驰成立] AND [C 段未破中枢区间]
→ 结论：其后必有回跌（或回升），方向回归中枢，但不必然产生大幅趋势性走势。

**情形2**：前提：[盘整背驰成立] AND [C 段已突破中枢区间（上破 ZG 或下破 ZD）] AND [回调不重新跌回中枢（上涨情形：回调不跌破 ZG）]
→ 结论：构成该级别的第三类买点（上涨方向）或第三类卖点（下跌方向），后续出现更高级别趋势性走势的概率极大。

**情形3**：前提：[盘整背驰成立] AND [C 段已突破中枢区间] AND [回调重新跌回中枢（上涨情形：跌破 ZG）]
→ 结论：不构成第三类买点，盘整继续。

> 原文引用："如果C段不破中枢，一旦出现MACD柱子的C段面积小于A段面积，其后必定有回跌。比较复杂的是如果C段上破中枢，但MACD柱子的面积小于A段的，这时候的原则是先出来，其后有两种情况，如果回跌不重新跌回，就在次级别的第一类买点回补，刚好这反而构成该级别的第三类买点，反之就继续该盘整。"
>
> ——课24（0436）

---

### 定理四：背驰后反弹强度区分定理（盘整背驰与趋势背驰最弱情形对比）

前提：[趋势背驰后发生最弱反弹（仅触及最后中枢 DD=min(dn)）]
→ 结论：反弹级别 ≥ 最后一个中枢级别。

前提：[盘整背驰后反弹，反弹不能重回中枢区间]
→ 结论：反弹级别 < 最后一个中枢级别（此为盘整背驰转化为第三类卖点的结构特征）。

> 原文引用："注意，这种情况和盘整背驰中转化成第三类卖点的情况不同，那种情况下，反弹的级别一定比最后一个中枢低，而这种情况，反弹的级别一定等于或大于最后一个中枢的。"
>
> ——课29（0456）

---

### 定理五：缠中说禅精确大转折点寻找程序定理（区间套）

前提：[已确认某大级别走势存在背驰段]
→ 结论：该大级别转折点可通过不同级别背驰段的逐级收缩区间而确定——在次级别图里找出对应背驰段，再在次次级别找，反复进行，直至最低操作级别，转折点在该级别背驰段确定的区间内。

> 原文引用："某大级别的转折点，先找到其背驰段，然后在次级别图里，找出相应背驰段在次级别里的背驰段，将该过程反复进行下去，直到最低级别，相应的转折点就在该级别背驰段确定的范围内。"
>
> ——课27（0449）

## 🔍 边界判断

### 盘整背驰 vs 趋势背驰 精确判断树

```
遇到力度衰减信号（C段面积 < A段面积，或C段未创新高/低）→

  步骤1：判断走势结构类型
    是否存在 ≥2 个同向、绝对不重叠的中枢（即趋势）？
      是 →
        步骤2：判断 A、B 中枢级别是否相同
          A、B 同级别？
            是 →
              步骤3：判断 c 段级别
                c 是否为次级别走势（包含对 B 的第三类买卖点）？
                  是 →
                    步骤4：判断 c 是否创新高/低
                      c 未创新高（上涨）/未创新低（下跌）
                        → 直接判断为趋势背驰，力度最弱，无需MACD辅助
                      c 创新高/低 →
                        比较 MACD 面积（c 段面积总和 vs a 段面积总和）：
                          c 面积 < a 面积 → 趋势背驰成立
                          c 面积 ≥ a 面积 → 趋势背驰不成立，走势继续延伸
                  否（c 级别过小，未包含 B 的三类买卖点）→
                    视为围绕 B 的次级别震荡，用盘整背驰方式判断
            否（A 级别大于 B）→
              整体视为围绕 A 的盘整震荡，进入「盘整背驰判断」
      否（单一中枢，盘整结构）→
        进入「盘整背驰判断」

  盘整背驰判断：
    存在 A（第一段）+ B（回调）+ C（第三段），C 方向与 A 同向？
      是 →
        C 面积（累加全部柱子）< A 面积，或 C 未创新高/低？
          是 → 盘整背驰成立
            → 进一步判断 C 段是否破中枢区间（上破 ZG / 下破 ZD）？
                C 未破中枢 → 必有回跌至中枢，不形成三类买卖点（定理三情形1）
                C 破中枢 → 先出场 → 观察回调：
                  回调不跌破 ZG（上涨方向）
                    → 第三类买点，在次级别第一类买点回补（定理三情形2）
                  回调跌破 ZG（上涨方向）
                    → 盘整继续，非第三类买点（定理三情形3）
          否 → 背驰不成立，走势延续
      否 → 结构不符合，重新判断级别或等待走势完成
```

### 趋势背驰内部结构约束（课37精确条件）

遇到疑似趋势背驰但结构模糊 →

  约束1：b 段级别须 ≤ c 段级别
    b 级别 > c 级别 → 趋势背驰极不标准，最弱走势，最终发生背驰则要特别留意后续

  约束2：c 段须包含 B 的第三类买卖点（即 c 至少包含两个次级别中枢）
    否则 → 不能视为标准趋势背驰的 c，用盘整背驰方式处理

  约束3：c 段须创新高（上涨趋势）或新低（下跌趋势）
    否 → 直接判定为最弱背驰，力度最差，无需 MACD 辅助确认

> 原文引用："c 必然是次级别的，也就是说，c 至少包含对 B 的一个第三类买卖点，否则，就可以看成是 B 中枢的小级别波动，完全可以用盘整背驰来处理。而 b 是有可能小于次级别的，力度最大的就是连续的缺口，也就是说，b 在级别上是不能大于 c 的。"
>
> ——课37（0481）

### 背驰与级别的两种转折方式区分

遇到背驰后的转折，判断其与当下走势级别的关系：

  情形一：背驰级别 = 当下走势级别
    → 转折必然发生，回跌（回升）至少进入最后一个中枢
    → 第一类买卖点价值最高

  情形二：背驰级别 < 当下走势级别
    → 转折不必然发生（可能形成次级别中枢后向突破方向继续延伸）
    → 须先形成比背驰级别更大的中枢，然后才可能形成回跌
    → 特别注意：出现在趋势冲顶或赶底的小级别背驰，往往引发大级别转折（不可忽视）

> 原文引用："第一种情况中，其回跌是必然的，而第二种情况，在形成一个比1分钟大的中枢后，并不必然回跌，可以往上突破，使得 `a+A+b+B+c` 继续延伸。"
>
> ——课43（0499）

### 大级别盘整背驰（周线以上）的特殊判断

遇到周线及以上级别的盘整背驰 →

  前提确认：站在最大级别看，绝大多数股票只有一个中枢（均为盘整结构）
  判断：此时盘整背驰构成「类第一类买点」，具有历史性大底意义
  级别要求：至少周线以上；月线/季线级别更佳

  MACD 判断标准（两条件满足其一即可，两条均满足为标准信号）：
    条件1：黄白线回抽 0 轴后再次下跌，未创新低
    条件2：C 段所有柱子面积之和 < A 段所有柱子面积之和

> 原文引用："盘整背驰最有用的，就是用在大级别上，特别是至少周线级别以上的，这种盘整背驰所发现的，往往就是历史性的大底部。"
>
> ——课27（0449）

## 💡 操作规则

### 操作规则一：趋势背驰——第一类买卖点介入规则

前提：[某级别趋势存在（≥2 个同向不重叠中枢，A、B 同级别）] AND [c 段出现趋势背驰信号（c 创新高/低 + MACD 面积 c < a，或 c 未创新高/低）]
触发：[背驰信号在 c 段内通过次级别结构确认（次级别内出现对应方向的背驰段）]
操作：[确认趋势背驰成立（区别于盘整背驰）] → [利用区间套从大级别向下逐级收缩找精确背驰点] → [在操作级别的第一类买（卖）点介入] → [止损设在最后一个中枢 ZD（上涨情形止损 ZD）之外]
冲突优先级：当同时出现疑似趋势背驰与盘整背驰信号时，须先判断 A、B 中枢是否同级别——同级别则用趋势背驰逻辑，A 级别大于 B 则让位于盘整背驰处理逻辑。

---

### 操作规则二：盘整背驰——C 段不破中枢出场规则

前提：[单一中枢盘整结构确认] AND [A+B+C 三段走势，C 方向与 A 同向] AND [C 段 MACD 面积 < A 段面积 且 C 段未突破中枢区间（ZG/ZD）]
触发：[C 段 MACD 背驰形态确认（累计所有柱子面积比较）]
操作：[确认盘整背驰成立] → [在 C 段 MACD 面积背驰信号出现时出场] → [等待价格回跌至中枢区间内出现的次级别第一类买点] → [回补（做短差）]
冲突优先级：若随后价格不回跌而继续突破，须重新判断是否演化为第三类买点（见规则三），届时本规则让位于规则三。

---

### 操作规则三：盘整背驰——C 段破中枢后观察第三类买卖点规则

前提：[单一中枢盘整结构确认] AND [C 段已突破中枢区间（上破 ZG 或下破 ZD）] AND [C 段 MACD 面积 < A 段面积，盘整背驰成立]
触发：[C 段破中枢后发生回调（上涨情形：回调启动）]
操作：[C 段破中枢出现盘整背驰信号时先出场] → [观察回调是否跌破 ZG（上涨情形）] →
  若回调不跌破 ZG：[在次级别第一类买点回补，此构成第三类买点，可持有至更高级别走势目标] →
  若回调跌破 ZG：[不回补，盘整继续，重新观察后续结构]
冲突优先级：第三类买点一旦确认，优先级高于其他短线操作；若不确认（跌破 ZG），本规则终止，回到规则二逻辑。

### 操作规则四：大级别盘整背驰——历史底部区间套定位规则

前提：[周线及以上级别确认盘整背驰（大级别单一中枢）] AND [C 段 MACD 面积明显 < A 段 AND/OR 黄白线未创新低]
触发：[大级别（月线/季线）背驰段确认（至少满足一个 MACD 标准）]
操作：[大级别背驰段确认后，向次级别（月线→周线→日线→30分钟→5分钟→1分钟）逐步寻找对应级别的背驰段] → [每次缩小区间] → [在操作级别找到第一类买点介入] → [止损设定宽（大级别确定性高，可越跌越买）]
冲突优先级：大级别盘整背驰构成「类第一类买点」，重要性高于小级别任何背驰信号；多级别综合考察（大级别强势时小级别背驰操作意义有限，甚至可以不操作）。

| 场景 | 判断倾向（原文指引） | 边界（不适用情形） |
|------|-------------------|--------------------|
| 趋势背驰的中枢数量规律 | 日线以上级别第二个中枢后背驰约占90%以上；小级别比例稍低但仍为大多数；4、5个中枢后才背驰相当罕见；100个中枢后才背驰百年一遇 | 延伸趋势中存在更多中枢的情况（极少见），不应以此规律作为唯一判断标准；实际需等待背驰信号出现后才操作 |
| MACD 图表选择：判断盘整背驰时应看哪个级别的图 | 用5分钟图判断盘整背驰一般比较清楚；A段与C段直接比较 | 若操作级别为更高级别（日线等），需相应选择该级别图表，不应固定使用5分钟图 |
| 盘整背驰嵌套：多次递减的盘整背驰 | 后者相对前者构成盘整背驰（g1d2 > g2d3 > g3d4），第二次红柱面积小于第一次时预示后面有麻烦 | 在1分钟图上这种背驰操作意义不大；在日线以上才有实质意义 |
| 背驰后惯性运动判断 | 背驰回来归根结底造成中枢震荡，震荡出第三类买卖点才有趋势的可能；不应预判"一定大幅下跌" | 大级别顶背驰中，小级别背驰可能引发暴跌；必须多级别综合考察，不可一刀切 |
| 多级别综合考察背驰操作意义 | 日线走势强势中段5分钟背驰操作意义有限（打短差甚至不操作）；日线走势最后阶段1分钟背驰足以引发暴跌 | 不能孤立看单一级别背驰就判断操作价值，必须先看高一级别走势状态 |
| 盘整背驰的层次地位 | 盘整背驰是对背驰（趋势背驰）力度分析方法的推广，主要用于中枢震荡的力度比较；趋势背驰是有基础意义的，盘整背驰是引申应用 | 不可将盘整背驰等同于趋势背驰——盘整背驰不保证回跌进入 B 段中枢，趋势背驰则有此保证 |
| 小级别盘整背驰在高位的操作意义 | 小级别高位盘整背驰意义不大，风险大，往往是"刀口舔血"；低位盘整背驰则构成多数的第二、三类买点 | 大级别（周线以上）盘整背驰具有历史性意义，不适用此限制 |
