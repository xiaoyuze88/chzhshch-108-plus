# P5-16-背驰的定义（均线面积法）

> **文件用途说明**：本文件是面向 AI 系统的缠论知识点文件，内容须自洽、无歧义、可机器解析。
> 所有定义优先使用充要条件格式，经验性判断须单独标注并与精确定义区分。

## 概述

背驰定义是缠论 P5-背驰理论中的核心构件，是第一类买卖点成立的充要前提。
本章定义了缠中说禅趋势力度（均线面积法）和背驰，提供了从"无趋势→无背驰"到"趋势结构→力度比较→背驰判定"的完整逻辑链，为 P6-买卖点理论的所有第一类买卖点操作提供前提。
原文在课15（初版/均线面积法核心定义）→ 课37（充要条件精细化）→ 课43（背驰段区分补充）间经历多次修订，核心定义以课15为基准，后续为补充说明。

## 📖 定义

### 定义一：缠中说禅趋势力度（均线面积法）

定义：当 [某段走势为趋势（含至少两个同向非重叠走势中枢）] 且 [该趋势中存在"吻"的序列（短线均线与长期均线发生相交）] 时，将 [前一"吻"的结束] 与 [后一"吻"的开始] 之间，由短线均线与长期均线相交所围成的面积，称为该段趋势的"缠中说禅趋势力度"；不在两次"吻"之间的走势段、以及盘整段中的均线缠绕，均不构成可度量的趋势力度单元。

> "缠中说禅趋势力度：前一'吻'的结束与后一'吻'开始由短线均线与长期均线相交所形成的面积。"
>
> ——课15（0402）

**补充约束（来自课15评论）**：
- 面积仅需目测对比，无需精确数值计算。
- 比较的是相邻两段趋势力度；中间如有盘整区，盘整不是趋势，跳过不计，不能跨越盘整进行非相邻比较。
- 形成趋势的才算面积；不断缠绕的均线是盘整，不能用于趋势力度计算。

---

### 定义二：背驰（均线面积法）——课15基准定义

定义：当且仅当满足以下全部条件时，构成背驰：
1. [存在趋势，即走势包含至少两个同向非重叠走势中枢]
2. [前后两段均可度量缠中说禅趋势力度（均存在"吻"序列）]
3. [后一段趋势力度（均线面积）严格小于前一段趋势力度]

不满足条件1（无趋势）则无背驰可言；不满足条件3（后段力度不弱于前段）则不构成背驰。

> "在前后两个同向趋势中，当缠中说禅趋势力度比上一次缠中说禅趋势力度要弱，就形成'背驰'。"
>
> ——课15（0402）

> "形成趋势后再谈背驰。对于一段趋势来说背驰只会一次。"
>
> ——课15（0402）评论

**核心约束**：
- 背驰只在趋势中比较，盘整中无背驰可言（盘整中的力度衰减另称"盘整背驰"，需严格区分）。
- 对于一段趋势，背驰只会出现一次。

### 定义三：缠中说禅趋势平均力度（即时判断法）

定义：当 [当前走势处于趋势中且尚未完成第二次"吻"] 时，将 [当下时刻] 与 [前一"吻"的结束] 之间，短线均线与长期均线所形成的面积除以时间，称为缠中说禅趋势平均力度；可即时（在第二次"吻"完成前）与前一次趋势平均力度作强弱对比，若当前值已弱于前一次，则可判断"背驰即将形成"。

> "缠中说禅趋势平均力度：当下与前一'吻'的结束时短线均线与长期均线形成的面积除以时间。因为这个概念是即时的，马上就可以判断当下的缠中说禅趋势平均力度与前一次缠中说禅趋势平均力度的强弱对比，一旦这次比上次弱，就可以判断'背驰'即将形成，然后再根据短线均线与长期均线的距离，一旦延伸长度缩短，就意味着真正的低部马上形成。"
>
> ——课15（0402）

**用途与限制**：用于在背驰完全确认前提前捕捉转折点（早于等"吻"完成后比较的方法），但技巧要求高，风险相对较大。

### 定义四：背驰充要条件精细化——课37三大前置条件（基于 a+A+b+B+c 结构）

定义：在 a+A+b+B+c 结构中，当且仅当同时满足以下三条件，才可讨论 c 段相对 a 段的趋势背驰；不满足任一条件则退化为盘整背驰或不构成背驰：

**条件一（趋势级别同一性）**：A 与 B 必须是同级别的走势中枢，且 a+A+b+B+c 整体构成趋势；若 A 级别大于 B，则结构退化为 a+A+(b+B+c)，只能用盘整背驰处理。

**条件二（c 的级别要求）**：c 必须是次级别走势，即 c 至少包含对 B 的一个第三类买卖点；否则 c 只是 B 中枢的小级别波动，用盘整背驰处理。b 的级别可小于次级别（最大力度为连续缺口），但 b 级别不能大于 c 级别。

**条件三（c 必须创新极值）**：若 a+A+b+B+c 为上涨，c 必须创出新高；若为下跌，c 必须创出新低；否则即使 c 包含 B 的第三类买卖点，也只能用盘整背驰方式处理。

> "没有趋势，没有背驰，不是任何 a+A+b+B+c 形式的都有背驰的。当说 a+A+b+B+c 中有背驰时，首先要 a+A+b+B+c 是一个趋势。而一个趋势，就意味着A、B是同级别的中枢……c 必然是次级别的，也就是说，c 至少包含对 B 的一个第三类买卖点……如果 a+A+b+B+c 是上涨，c 一定要创出新高；a+A+b+B+c 是下跌，c 一定要创出新低。"
>
> ——课37（0481）

**附加推论（来自课37）**：c 内部结构至少含两个次级别中枢（构成次级别趋势），可递归套用 a+A+b+B+c 形式进行次级别背驰分析，形成区间套定位。

### 定义五：背驰段（进入背驰段 vs 背驰成立）

定义：当 [某级别走势已进入可能的背驰对比段（力度指标初步显示后段弱于前段）] 但 [该段走势尚未完成] 时，称为"进入背驰段"；进入背驰段不等同于背驰已成立——若后续次级别走势足够强，使得均线黄白线创出新高（或新低），则背驰段最终不成立，不构成背驰。

> "这个背驰段并不一定就演化成背驰，因为，小级别的延伸足以使得大级别最终摆脱背驰……如果用 MACD 来辅助看，拉回0轴后再上去，都可以先看成是进入背驰段……但最后是否黄白线创新高，在刚走的时候是不可能知道的，然后就要看小级别的结构，如果小级别的走势特别强，使得黄白线创出新高，那就不存在背驰的问题了。"
>
> ——课43（0499）

## 📐 定理

### 定理一：没有趋势，没有背驰（核心原则）

前提：[某级别走势不包含至少两个同向非重叠走势中枢，即不构成趋势]
→ 结论：[该级别走势中无背驰可言；任何力度衰减只能称为盘整背驰，与趋势背驰性质不同，需严格区分]

> "必须注意：没有趋势，没有背驰。在盘整中是无所谓'背驰'的，这点是必须特别明确的。"
>
> ——课15（0402）

> "没有趋势没有背驰，背驰是两个趋势之间比较才有意义，和盘整里比较是没用的。背驰是两个前后趋势之间的比较。"
>
> ——课16（0408）

---

### 定理二：一段趋势背驰只出现一次

前提：[确定一个完整的同级别趋势（含至少两个同向非重叠中枢）]
→ 结论：[该趋势中的背驰只会出现一次；若观察到同一趋势中多次背驰，则说明背驰概念定义有误，或误将盘整段纳入了比较]

> "对于一段趋势来说背驰只会一次。"
>
> ——课15（0402）评论

> "背驰只可能出现一次，怎么可能一次又一次。你认为的一次又一次的，根本就不是本ID所说的背驰，注意，背驰是两个同级别趋势之间对比产生的。"
>
> ——课17（0412）

---

### 定理三：背驰导致转折（背驰-转折定理）

前提：[某级别趋势中出现该级别背驰]
→ 结论：[该趋势必然发生转折——要么该级别最后一个中枢扩展为更高级别中枢，要么产生该级别以上级别的反趋势；背驰后逆转是无条件必然的，但逆转的级别与持续时间取决于背驰的级别]

> "只要你看到某级别的背驰，必然意味着要有逆转。"
>
> ——课24（0436）

> "某级别趋势的背驰将导致该趋势最后一个中枢的级别扩展、该级别更大级别的盘整或该级别以上级别的反趋势。"
>
> ——课43（0499）引用课29定理

---

### 定理四：背驰-买卖点定理

前提：[某级别走势出现背驰]
→ 结论：[该背驰必然制造某级别的买卖点；反之，任一级别的买卖点都必然源自某级别走势的背驰]

> "缠中说禅背驰-买卖点定理：任一背驰都必然制造某级别的买卖点，任一级别的买卖点都必然源自某级别走势的背驰。"
>
> ——课24（0436）

---

### 定理五：任何走势结束必因背驰

前提：[任何级别的任何走势类型完成]
→ 结论：[该走势的结束必然由某级别的背驰导致；该级别背驰至少结束该级别的走势，但不一定是次级别背驰导致]

> "任何走势的结束，一定由于背驰，某级别的背驰，至少结束该级别的走势，这是没问题。"
>
> ——课33（0469）

---

### 定理六：背驰级别等于当下走势级别时的必然回拉

前提：[某级别走势类型出现等级别背驰（背驰级别 = 当下走势级别）]
→ 结论：[该背驰至少把走势拉向该走势类型最后一个中枢；回拉是必然的]

> "背驰级别等于当下的走势级别，例如，一个30分钟的走势类型，出现一个30分钟级别的背驰，那么这个背驰至少把走势拉向该30分钟走势最后一个中枢。"
>
> ——课43（0499）

---

### 定理七：背驰级别小于当下走势级别时的非必然性

前提：[某级别走势类型中出现小级别背驰（背驰级别 < 当下走势级别），且当下走势明确无该级别背驰]
→ 结论：[小级别背驰导致的回调并不必然跌回最后中枢；回调必须先形成比该背驰级别更大的中枢，才可能进一步向下延伸；也可能直接往上突破使走势继续延伸]

> "这第二种情况，必然要先形成一个比1分钟级别要大的中枢，然后向下突破，最终形成回跌到B中的走势。"
>
> ——课43（0499）

## 🔍 边界判断

### 有无趋势的判断树（背驰前置判断）

遇到"是否有背驰"的问题 →
  先判断"是否存在趋势（至少两个同向非重叠中枢）"：
    否 → **无背驰可言**，只能讨论盘整背驰。停止背驰判断。
    是 → 判断"是否为 a+A+b+B+c 结构且已完成两次吻" →
      是 → 继续课37三大条件判断（见下方决策树）
      结构完整且已完成两次"吻" → 直接对比相邻两段趋势力度（均线面积），后段弱于前段则确认背驰

---

### 课37三大条件决策树（a+A+b+B+c 趋势背驰 vs 盘整背驰判断）

遇到 a+A+b+B+c 结构，判断是否构成趋势背驰 →

  **步骤1：判断条件一（A、B同级别）**：
    A 级别 = B 级别，且整体为趋势（两中枢无重叠，同向） → 通过，进入步骤2
    A 级别 > B 级别 → 退化：a+A+(b+B+c)，最多讨论盘整背驰，停止趋势背驰判断

  **步骤2：判断条件二（c 的级别要求）**：
    c 至少包含对 B 的第三类买卖点（c 是次级别） → 通过，进入步骤3
    c 不包含 B 的第三类买卖点 → 退化为盘整背驰，停止趋势背驰判断

  **步骤3：判断条件三（c 是否创新极值）**：
    上涨结构：c 创新高 → 通过，进入步骤4
    下跌结构：c 创新低 → 通过，进入步骤4
    未创新高/新低 → 退化为盘整背驰处理，停止趋势背驰判断

  **步骤4：趋势力度比较（均线面积法）**：
    c 段均线面积 < a 段均线面积 → 构成背驰
    c 段均线面积 ≥ a 段均线面积 → 不构成背驰，走势延续

---

### 背驰 vs 盘整背驰 区分

| 维度 | 趋势背驰 | 盘整背驰 |
|------|---------|---------|
| 前提结构 | 至少两个同向非重叠中枢（趋势） | 单一中枢（盘整） |
| 比较对象 | 前后两段离开中枢的趋势力度（均线面积） | 围绕同一中枢的两次同向离开段力度 |
| 结果必然性 | 必然导致该级别走势转折（定理三） | 多数情况下最终突破或形成第三类买卖点（无固定概率，90%经验数字指"趋势背驰发生在第二中枢后"，与此不同） |
| 课37条件 | 需满足三大精细条件（级别/次级别/创新极值） | 无需三大条件，只需同一中枢内比较 |
| 操作含义 | 第一类买卖点前提 | 需结合中枢突破方向判断，见 P5-17 |

### 特殊情况处理

**特殊情况一：进入背驰段但背驰未成立**

遇到"已进入背驰段，但走势继续延伸" →
  判断"次级别走势是否足够强（MACD 黄白线是否创新高/新低）"：
    是（黄白线创新高/新低）→ 背驰段不成立，原趋势延续，重新计算趋势力度
    否（黄白线未创新高）→ 结合价格走势结构，等待吻的形成以最终确认背驰

**特殊情况二：小级别背驰引发大级别转折**

遇到"当下走势级别无背驰，但小级别出现背驰" →
  判断"小级别背驰后是否形成更大中枢 C，且 C 与 B 的重叠关系" →
    C 与 B 无重叠 → 走势延伸为 a+A+b+B+c+C，原走势类型未完成
    C 与 B 有重叠 → (B+c+C) 扩张为更高级别中枢，按走势分解原则重新归类

**特殊情况三：a+A+b+B+c 中 b 为连续缺口（极小级别）**

遇到"b 为连续缺口，级别极小" →
  判断"c 是否为次级别且创新高/新低" →
    是 → 可继续判断趋势背驰，但须注意：b 如此之小而 c 延伸成次级别，c 最终出现背驰的可能性极小；即使背驰，极有可能演化为最弱走势（参见📜经验判断）
    否 → 退化为盘整背驰处理

**特殊情况四：a+A+b 与 c 的反向力度比较（趋势结束后的反转走势判断）**

遇到"需要判断趋势末端 c 段之后、反向走势 c 相对原趋势 a+A+b 的力度" →
  此为反方向的力度比较，与同向背驰判断的区别：
    同向背驰：MACD 黄白线和柱子面积都在0轴同一侧（都在上方或都在下方）
    反向比较（a+A+b vs c）：两者分别在0轴两侧，黄白线至少穿越一次0轴，不存在黄白线回拉0轴问题
  参见课33（0469）

## 💡 操作规则

### 操作规则一：均线面积法背驰确认规则（标准方法）

前提：[当前级别走势已确认为趋势（A、B 同级别且无重叠）] AND [c 段满足课37三大条件（同级别中枢、c 是次级别、c 创新极值）]
触发：[c 段第二次"吻"已形成（短线均线再次穿越长期均线），可对比前后两段面积]
操作：[目测估算 c 段均线面积（前一"吻"结束至后一"吻"开始之间）] → [与 a 段对应面积对比] → [若 c 段面积 < a 段面积，确认背驰成立] → [依据背驰级别进入第一类买卖点操作逻辑（见 P6-20/P6-21）]
冲突优先级：当 [同时存在多级别背驰信号] 时，本规则 [让位于] [更高级别背驰信号]，原因：级别越大对应的买卖点意义越重要

---

### 操作规则二：趋势平均力度即时判断规则（高阶方法）

前提：[当前级别走势已确认为趋势] AND [第一段趋势（a）已完成且形成吻] AND [第二段趋势（c）尚未完成第二次吻]
触发：[实时计算：(当下 ~ 前一吻结束) 的均线面积 ÷ 时间，与前一次趋势平均力度对比，当前值已弱于前值] AND [短线均线与长期均线的距离开始缩短]
操作：[判断"背驰即将形成"] → [密切关注 c 段内次级别结构的反转信号] → [在次级别第一类买卖点出现时介入]
冲突优先级：当 [均线距离短暂缩短但随后再次扩张（背驰段未成立）] 时，本规则 [让位于] [规则一（等吻完成后确认法）]，原因：趋势平均力度仅为预判，背驰段不等于背驰，须等待最终确认

---

### 操作规则三：背驰判断前置检验规则（防止误判）

前提：[观察到某走势中出现疑似力度衰减信号]
触发：[准备判断背驰之前]
操作：[先确认"是否存在趋势"（至少两个同向非重叠中枢）] → [若无趋势，停止背驰判断，改用盘整背驰框架（见 P5-17）] → [若有趋势，核查课37三大条件（A/B同级别、c 为次级别、c 创新极值）] → [三条均满足，方可进入趋势力度比较]
冲突优先级：本规则为所有背驰判断的前置强制步骤，无例外；任何跳过此规则直接看 MACD 或均线信号的操作均视为无效判断

## 📜 经验判断

> ⚠️ **原文未给出充要条件**，以下为原文经验性指引，AI 使用时须视为参考倾向而非确定规则。

| 场景 | 判断倾向（原文指引） | 边界（不适用情形） |
|------|-------------------|--------------------|
| 场景：[b 为连续缺口（极小级别），c 延伸至次级别后进入背驰段] | 判断倾向：[c 最终形成背驰的可能性很小；即使背驰，后续极有可能出现最弱走势] | 边界：[若 c 次级别内部结构强劲，黄白线未出现弱化，则不能简单认定背驰，需等待确认] |
| 场景：[盘整背驰后，走势未演化为 a+B' 继续原方向] | 判断倾向：[必然跌破或升破该盘整区间；多数情况下也会突破，但不具有绝对必然性] | 边界：[存在一定概率不突破而继续盘整，不可当成100%必然；尤其高级别中枢震荡时慎用] |
| 场景：[小级别背驰出现，大级别尚无背驰信号] | 判断倾向：[先形成比小级别更大的中枢，然后判断该中枢与原结构的重叠关系，再决定是否延伸或转折] | 边界：[若大级别趋势特别强，小级别背驰后往往直接被消化，不形成大转折] |
| 场景：[用 MACD 辅助看背驰（黄白线 + 红绿柱面积）] | 判断倾向：[MACD 是均线面积法的近似简化，可辅助目测对比；拉回0轴后再上去可先看成进入背驰段] | 边界：[MACD 不能替代走势结构分析；必须先完成走势分解确认趋势和中枢，再看对应级别 MACD；跨级别看 MACD 造成误判] |
| 场景：[一切转折的起点判断] | 判断倾向：["一切转折，基本都是从'湿吻'开始的"（课15）；飞吻和唇吻力度不足，湿吻标志力度达到转折临界] | 边界：[湿吻是转折的必要但非充分条件，还需看背驰是否成立] |
